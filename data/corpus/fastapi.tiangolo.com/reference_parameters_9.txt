Title: Deprecation of kwargs in FastAPI Parameters
URL: https://fastapi.tiangolo.com/reference/parameters/
Summary: The `kwargs` parameter is deprecated in FastAPI. Users should now utilize `json_schema_extra` to include additional fields for JSON Schema, with the type set to `Any` and default as an empty dictionary.
