Title: Separate OpenAPI Schemas for Input and Output
URL: https://fastapi.tiangolo.com/how-to/separate-openapi-schemas/
Summary: This document discusses the use of separate Pydantic models for input and output in FastAPI to enhance API documentation and maintain clarity in data handling. It covers creating models for both input and output, how these models appear in documentation, and the benefits of distinguishing between input and output schemas.
