# Path Parameters[Â¶](https://fastapi.tiangolo.com/tutorial/path-params/#path-parameters "Permanent link")
You can declare path "parameters" or "variables" with the same syntax used by Python format strings:
[Python 3.8+](https://fastapi.tiangolo.com/tutorial/path-params/#__tabbed_1_1)
```
fromfastapiimport FastAPI
app = FastAPI()
@app.get("/items/{item_id}")
async defread_item(item_id):
  return {"item_id": item_id}

```

The value of the path parameter `item_id` will be passed to your function as the argument `item_id`.
So, if you run this example and go to <http://127.0.0.1:8000/items/foo>, you will see a response of:
```
{"item_id":"foo"}
