Title: FastAPI Release Notes 0.82.0
URL: https://fastapi.tiangolo.com/release-notes/
Summary: This release marks potentially the last version supporting Python 3.6, which is now end-of-life. Key features include exporting `WebSocketState`, supporting Python internal descriptions in Pydantic model docstrings, and updating `ORJSONResponse` for better compatibility with non-string keys and Numpy arrays.
