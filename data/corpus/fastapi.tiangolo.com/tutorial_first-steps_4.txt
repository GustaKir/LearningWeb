The simplest FastAPI file could look like this:
[Python 3.8+](https://fastapi.tiangolo.com/tutorial/first-steps/#__tabbed_1_1)
```
fromfastapiimport FastAPI
app = FastAPI()
@app.get("/")
async defroot():
  return {"message": "Hello World"}

```

Copy that to a file `main.py`.
