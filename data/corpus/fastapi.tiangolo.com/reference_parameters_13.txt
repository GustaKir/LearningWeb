Title: FastAPI Body Parameter Configuration
URL: https://fastapi.tiangolo.com/reference/parameters/
Summary: This section describes the configuration options for body parameters in FastAPI, including 'default', 'default_factory', 'embed', and 'media_type'. It explains the purpose of each option and how they affect the behavior of parameters in JSON requests.
