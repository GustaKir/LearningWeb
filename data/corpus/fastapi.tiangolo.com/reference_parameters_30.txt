Title: Parameters Reference in FastAPI
URL: https://fastapi.tiangolo.com/reference/parameters/
Summary: This section discusses various parameter attributes in FastAPI, such as 'deprecated' to mark parameters as outdated and 'include_in_schema' to control parameter visibility in the OpenAPI documentation. It highlights the enhanced support for OpenAPI-specific examples in Swagger UI and includes a link to further documentation on declaring request example data.
