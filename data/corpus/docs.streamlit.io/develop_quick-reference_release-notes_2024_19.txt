Title: Streamlit Release Notes - Notable Changes for 2024
URL: https://docs.streamlit.io/develop/quick-reference/release-notes/2024
Summary: This document highlights key updates in Streamlit's 2024 release, including configurable heights for `st.container`, a simplified UI for `LinkColumn` in dataframes, the ability to hide sidebar navigation in multipage apps, and improved loading speeds for Plotly figures using `orjson`. It also notes a behavioral change in an unspecified feature.
