## [](https://docs.streamlit.io/develop/quick-reference/release-notes/2024#version-1400)**Version 1.40.0**
_Release date: November 6, 2024_
**Highlights**
  * 💊 Introducing [`st.pills`](https://docs.streamlit.io/develop/api-reference/widgets/st.pills) to create a single- or multi-select group of pill-buttons.
  * 🎛️ Introducing [`st.segmented_control`](https://docs.streamlit.io/develop/api-reference/widgets/st.segmented_control) to create a segmented button or button group.
  * 🎤 Announcing the general availability of [`st.audio_input`](https://docs.streamlit.io/\[...slug\]), a widget to let users record sound with their microphones.


**Notable Changes**
  * ➡️ Markdown renders a limited set of typographical symbols (arrows and comparators).
  * ![](https://docs.streamlit.io/logo.svg) You can use `:streamlit:` to render the Streamlit logo in [Markdown](https://docs.streamlit.io/develop/api-reference/text/st.markdown).
  * 🐍 [`st.text`](https://docs.streamlit.io/develop/api-reference/text/st.text) wraps text and no longer uses monospace font.
  * 🪣 You can set `use_container_width` for [`st.image`](https://docs.streamlit.io/develop/api-reference/media/st.image). `use_column_width` is deprecated.
  * 📅 [`st.date_input`](https://docs.streamlit.io/develop/api-reference/widgets/st.date_input) infers the first day of the week from the user’s locale ([#9706](https://github.com/streamlit/streamlit/pull/9706), [#5215](https://github.com/streamlit/streamlit/issues/5215)).

  * 🚀 Streamlit provides helpful links for deployment when an app is running locally ([#9681](https://github.com/streamlit/streamlit/pull/9681)).
  * ↕️ The fullscreen button for charts matches with the dataframe toolbar ([#9721](https://github.com/streamlit/streamlit/pull/9721)).
