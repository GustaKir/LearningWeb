Title: Streamlit 2024 Release Notes - Other Changes
URL: https://docs.streamlit.io/develop/quick-reference/release-notes/2024
Summary: This segment highlights various changes in Streamlit's 2024 release, including improvements such as lazy-loading for modules to enhance import speed, support for PIL images in `st.write`, the addition of markdown links in `st.radio` options, and the deprecation of the `deprecation.showPyplotGlobalUse` configuration option.
