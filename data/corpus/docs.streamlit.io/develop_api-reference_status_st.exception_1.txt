Title: st.exception: Displaying Exceptions in Streamlit
URL: https://docs.streamlit.io/develop/api-reference/status/st.exception
Summary: The st.exception function in Streamlit is used to display exceptions in the interface. It includes links to Google and ChatGPT in the lower-right corner, prefilled with the exception message content, facilitating quick access to troubleshooting resources.

## [](https://docs.streamlit.io/develop/api-reference/status/st.exception#stexception)st.exception
Streamlit VersionVersion 1.44.0Version 1.43.0Version 1.42.0Version 1.41.0Version 1.40.0Version 1.39.0Version 1.38.0Version 1.37.0Version 1.36.0Version 1.35.0Version 1.34.0Version 1.33.0Version 1.32.0Version 1.31.0Version 1.30.0Version 1.29.0Version 1.28.0Version 1.27.0Version 1.26.0Version 1.25.0Version 1.24.0Version 1.23.0Version 1.22.0Version 1.21.0Version 1.20.0
Display an exception.
In the lower-right corner of the exception, Streamlit displays links to Google and ChatGPT that are prefilled with the contents of the exception message.
Function signature[[source]](https://github.com/streamlit/streamlit/blob/1.44.0/lib/streamlit/elements/exception.py#L42 "View st.exception source code on GitHub")  
---  
st.exception(exception)  
Parameters  
exception (Exception) |  The exception to display.  
#### Example
```
import streamlit as st
e = RuntimeError("This is an exception of type RuntimeError")
st.exception(e)
```
