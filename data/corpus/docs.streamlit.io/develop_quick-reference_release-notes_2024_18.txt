Title: Streamlit Release Notes - Version 1.31.0
URL: https://docs.streamlit.io/develop/quick-reference/release-notes/2024
Summary: The 1.31.0 release of Streamlit includes new features such as 'st.page_link' for custom navigation in multipage apps and 'st.write_stream' for easier handling of data streams. Several bug fixes were also addressed, improving compatibility with dark mode and correcting display issues in Chrome.
