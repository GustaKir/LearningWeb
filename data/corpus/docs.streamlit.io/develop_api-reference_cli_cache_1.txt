Title: Streamlit CLI: Cache Clear Command
URL: https://docs.streamlit.io/develop/api-reference/cli/cache
Summary: The `$ streamlit cache clear` command is used to remove persisted files from the Streamlit cache on disk. This section includes the syntax for the command and links to related topics in the Streamlit documentation.
---

## [](https://docs.streamlit.io/develop/api-reference/cli/cache#-streamlit-cache-clear)`$ streamlit cache clear`
Clear persisted files from the on-disk [Streamlit cache](https://docs.streamlit.io/develop/api-reference/caching-and-state), if present.
### [](https://docs.streamlit.io/develop/api-reference/cli/cache#syntax)Syntax
`streamlit cache clear `
