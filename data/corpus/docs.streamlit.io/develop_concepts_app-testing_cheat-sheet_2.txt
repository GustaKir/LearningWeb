Title: Streamlit App Testing: Supported Elements
URL: https://docs.streamlit.io/develop/concepts/app-testing/cheat-sheet
Summary: This section outlines the support for various Streamlit UI elements in app testing using `AppTest.get()`. It includes chart elements, media elements, file uploader, data editor, expander, status, camera input, download button, and link button. The document mentions plans to regularly enhance support for additional features.
