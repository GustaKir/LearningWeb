Title: Streamlit Radio Widget Customization
URL: https://docs.streamlit.io/develop/api-reference/widgets/st.radio
Summary: This section outlines how to customize the label visibility of radio widgets in Streamlit using the `label_visibility` parameter. It explains the options available: 'hidden' maintains space for the label, 'collapsed' removes both the label and space, while 'visible' shows the label. Additionally, it covers the use of the `disabled` parameter to disable the widget and the `horizontal` parameter to arrange options horizontally.
