# [](https://docs.streamlit.io/develop/api-reference/cli#command-line-interface)Command-line interface
When you install Streamlit, a command-line (CLI) tool gets installed as well. The purpose of this tool is to run Streamlit apps, change Streamlit configuration options, and help you diagnose and fix issues.
## [](https://docs.streamlit.io/develop/api-reference/cli#available-commands)Available commands
  * [`streamlit cache clear`](https://docs.streamlit.io/develop/api-reference/cli/cache): Clear the on-disk cache.
  * [`streamlit config show`](https://docs.streamlit.io/develop/api-reference/cli/config): Show all configuration options.
  * [`streamlit docs`](https://docs.streamlit.io/develop/api-reference/cli/docs): Open the Streamlit docs.
  * [`streamlit hello`](https://docs.streamlit.io/develop/api-reference/cli/hello): Run an example Streamlit app.
  * [`streamlit help`](https://docs.streamlit.io/develop/api-reference/cli/help): Show the available CLI commands.
  * [`streamlit init`](https://docs.streamlit.io/develop/api-reference/cli/init): Create the files for a new Streamlit app.
  * [`streamlit run`](https://docs.streamlit.io/develop/api-reference/cli/run): Run your Streamlit app.
  * [`streamlit version`](https://docs.streamlit.io/develop/api-reference/cli/version): Show the version of Streamlit.


### [](https://docs.streamlit.io/develop/api-reference/cli#run-your-app)Run your app
The most important command is `streamlit run`, which is summarized for convenience here:
`streamlit run your_script.py `
At any time, in your terminal, you can stop the server with **Ctrl+C**.
