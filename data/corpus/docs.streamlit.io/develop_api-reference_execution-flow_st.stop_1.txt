Title: st.stop - Streamlit Execution Control
URL: https://docs.streamlit.io/develop/api-reference/execution-flow/st.stop
Summary: The st.stop() function in Streamlit halts the execution of the script immediately, preventing any subsequent code from running. It is advised to render a message to inform users why the script has stopped. An example illustrates its use by prompting for user input and warning if none is provided before stopping execution.

Stops execution immediately.
Streamlit will not run any statements after st.stop(). We recommend rendering a message to explain why the script has stopped.
Function signature[[source]](https://github.com/streamlit/streamlit/blob/1.44.0/lib/streamlit/commands/execution_control.py#L34 "View st.stop source code on GitHub")  
---  
st.stop()  
#### Example
```
import streamlit as st
name = st.text_input("Name")
if not name:
 st.warning('Please input a name.')
 st.stop()
st.success("Thank you for inputting a name.")
```
