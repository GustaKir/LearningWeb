## [](https://docs.streamlit.io/develop/concepts/custom-components/limitations#how-do-streamlit-components-differ-from-functionality-provided-in-the-base-streamlit-package)How do Streamlit Components differ from functionality provided in the base Streamlit package?
  * Streamlit Components are wrapped up in an iframe, which gives you the ability to do whatever you want (within the iframe) using any web technology you like.
