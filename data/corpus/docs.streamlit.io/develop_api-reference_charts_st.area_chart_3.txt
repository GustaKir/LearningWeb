Title: Creating a Vega-Lite Chart in Streamlit
URL: https://docs.streamlit.io/develop/api-reference/charts/st.area_chart
Summary: This chunk demonstrates how to create a Vega-Lite chart in Streamlit using a specified dataset. It shows how to define a line chart with encoding for the x and y axes, and how to add rows to the chart from a named dataset.
---

```
my_chart = st.vega_lite_chart(
  {
    "mark": "line",
    "encoding": {"x": "a", "y": "b"},
    "datasets": {
      "some_fancy_name": df1, # <-- named dataset
    },
    "data": {"name": "some_fancy_name"},
  }
)
my_chart.add_rows(some_fancy_name=df2) # <-- name used as keyword
```
