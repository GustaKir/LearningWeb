## [](https://docs.streamlit.io/develop/api-reference/caching-and-state/st.context#stcontext)st.context
Streamlit VersionVersion 1.44.0Version 1.43.0Version 1.42.0Version 1.41.0Version 1.40.0Version 1.39.0Version 1.38.0Version 1.37.0Version 1.36.0Version 1.35.0Version 1.34.0Version 1.33.0Version 1.32.0Version 1.31.0Version 1.30.0Version 1.29.0Version 1.28.0Version 1.27.0Version 1.26.0Version 1.25.0Version 1.24.0Version 1.23.0Version 1.22.0Version 1.21.0Version 1.20.0
An interface to access user session context.
`st.context` provides a read-only interface to access headers and cookies for the current user session.
Each property (`st.context.headers` and `st.context.cookies`) returns a dictionary of named values.

## [](https://docs.streamlit.io/develop/api-reference/caching-and-state/st.context#contextheaders)context.headers

A read-only, dict-like object containing headers sent in the initial request.
Keys are case-insensitive and may be repeated. When keys are repeated, dict-like methods will only return the last instance of each key. Use `.get_all(key="your_repeated_key")` to see all values if the same header is set multiple times.
Function signature[[source]](https://github.com/streamlit/streamlit/blob/1.44.0/lib/streamlit/runtime/context.py#L137 "View st.headers source code on GitHub")
