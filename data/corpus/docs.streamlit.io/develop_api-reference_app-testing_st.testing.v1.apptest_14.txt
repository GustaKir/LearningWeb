Title: Accessing Individual Widgets in Streamlit's App Testing
URL: https://docs.streamlit.io/develop/api-reference/app-testing/st.testing.v1.apptest
Summary: This document describes how to access individual widgets from a WidgetList by index or key in Streamlit's app testing module. It includes specific references to the `at.text_input` method and introduces the `AppTest.time_input` function that returns a sequence of all `st.time_input` widgets.
---

Individual widgets can be accessed from a WidgetList by index (order on the page) or key. For example, `at.text_input[0]` for the first widget or `at.text_input(key="my_key")` for a widget with a given key.
