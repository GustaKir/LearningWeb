## [](https://docs.streamlit.io/develop/api-reference/cli/config#-streamlit-config-show)`$ streamlit config show`
Print all the available configuration options, including their descriptions, default values, and current values. For more information about configuration options, see [`config.toml`](https://docs.streamlit.io/develop/api-reference/configuration/config.toml).
### [](https://docs.streamlit.io/develop/api-reference/cli/config#syntax)Syntax
`streamlit config show `
