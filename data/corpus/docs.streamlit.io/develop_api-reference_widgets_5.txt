[![screenshot](https://docs.streamlit.io/images/api/components/stqdm.jpg)StqdmThe simplest way to handle a progress bar in streamlit app. Created by [@Wirg](https://github.com/Wirg).`from stqdm import stqdm for _ in stqdm(range(50)):   sleep(0.5) `](https://github.com/Wirg/stqdm)
[![screenshot](https://docs.streamlit.io/images/api/components/timeline.jpg)TimelineDisplay a Timeline in Streamlit apps using [TimelineJS](https://timeline.knightlab.com/). Created by [@innerdoc](https://github.com/innerdoc).`from streamlit_timeline import timeline with open('example.json', "r") as f:  timeline(f.read(), height=800) `](https://github.com/innerdoc/streamlit-timeline)
[![screenshot](https://docs.streamlit.io/images/api/components/camera-live.jpg)Camera input liveAlternative for st.camera_input which returns the webcam images live. Created by [@blackary](https://github.com/blackary).`from camera_input_live import camera_input_live image = camera_input_live() st.image(value) `](https://github.com/blackary/streamlit-camera-input-live)
[![screenshot](https://docs.streamlit.io/images/api/components/ace.jpg)Streamlit AceAce editor component for Streamlit. Created by [@okld](https://github.com/okld).`from streamlit_ace import st_ace content = st_ace() content `](https://github.com/okld/streamlit-ace)
[![screenshot](https://docs.streamlit.io/images/api/components/chat.jpg)Streamlit ChatStreamlit Component for a Chatbot UI. Created by [@AI-Yash](https://github.com/AI-Yash).`from streamlit_chat import message message("My message") message("Hello bot!", is_user=True) # align's the message to the right `](https://github.com/AI-Yash/st-chat)
[![screenshot](https://docs.streamlit.io/images/api/components/option-menu.jpg)Streamlit Option MenuSelect a single item from a list of options in a menu.
