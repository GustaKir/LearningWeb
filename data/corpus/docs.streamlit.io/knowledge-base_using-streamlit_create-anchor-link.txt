# [](https://docs.streamlit.io/knowledge-base/using-streamlit/create-anchor-link#how-do-i-create-an-anchor-link)How do I create an anchor link?
## [](https://docs.streamlit.io/knowledge-base/using-streamlit/create-anchor-link#overview)Overview
Have you wanted to create anchors so that users of your app can directly navigate to specific sections by specifying `#anchor` in the URL? If so, let's find out how.
## [](https://docs.streamlit.io/knowledge-base/using-streamlit/create-anchor-link#solution)Solution
Anchors are automatically added to header text.
For example, if you define a header text via the [st.header()](https://docs.streamlit.io/develop/api-reference/text/st.header) command as follows:
`st.header("Section 1") `
Then you can create a link to this header using:
`st.markdown("[Section 1](#section-1)") `
