Title: Increasing File Uploader Limit on Streamlit Community Cloud
URL: https://docs.streamlit.io/knowledge-base/deploy/increase-file-uploader-limit-streamlit-cloud
Summary: To increase the file upload limit when deploying to Streamlit Community Cloud, use option 1 by setting the `maxUploadSize` in a global config file (`.streamlit/config.toml`). For example, for a 400MB limit, use `[server] maxUploadSize = 400` in the config file uploaded to your app's GitHub repository.
---

## [](https://docs.streamlit.io/knowledge-base/deploy/increase-file-uploader-limit-streamlit-cloud#solution)Solution
When deploying your app to Streamlit Community Cloud, you should **use option 1**. Namely, set the `maxUploadSize` config option in a global config file (`.streamlit/config.toml`) uploaded to your app's GitHub repo. ðŸŽˆ
For example, to increase the upload limit to 400MB, upload a `.streamlit/config.toml` file containing the following lines to your app's GitHub repo:
`[server] maxUploadSize = 400 `
