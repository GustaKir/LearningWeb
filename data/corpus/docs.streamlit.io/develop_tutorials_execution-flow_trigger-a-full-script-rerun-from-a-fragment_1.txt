# [](https://docs.streamlit.io/develop/tutorials/execution-flow/trigger-a-full-script-rerun-from-a-fragment#trigger-a-full-script-rerun-from-inside-a-fragment)Trigger a full-script rerun from inside a fragment
Streamlit lets you turn functions into [fragments](https://docs.streamlit.io/develop/concepts/architecture/fragments), which can rerun independently from the full script. When a user interacts with a widget inside a fragment, only the fragment reruns. Sometimes, you may want to trigger a full-script rerun from inside a fragment. To do this, call [`st.rerun`](https://docs.streamlit.io/develop/api-reference/execution-flow/st.rerun) inside the fragment.
## [](https://docs.streamlit.io/develop/tutorials/execution-flow/trigger-a-full-script-rerun-from-a-fragment#applied-concepts)Applied concepts
  * Use a fragment to rerun part or all of your app, depending on user input.


## [](https://docs.streamlit.io/develop/tutorials/execution-flow/trigger-a-full-script-rerun-from-a-fragment#prerequisites)Prerequisites
  * This tutorial requires the following version of Streamlit:
`streamlit>=1.37.0 `
  * You should have a clean working directory called `your-repository`.
  * You should have a basic understanding of fragments and `st.rerun`.
