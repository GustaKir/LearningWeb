# [](https://docs.streamlit.io/knowledge-base/using-streamlit/how-download-pandas-dataframe-csv#how-to-download-a-pandas-dataframe-as-a-csv)How to download a Pandas DataFrame as a CSV?
Use the [`st.download_button`](https://docs.streamlit.io/develop/api-reference/widgets/st.download_button) widget that is natively built into Streamlit. Check out a [sample app](https://streamlit-release-demos-0-88streamlit-app-0-88-v8ram3.streamlit.app/) demonstrating how you can use `st.download_button` to download common file formats.
## [](https://docs.streamlit.io/knowledge-base/using-streamlit/how-download-pandas-dataframe-csv#example-usage)Example usage
`import streamlit as st import pandas as pd df = pd.read_csv("dir/file.csv") @st.cache_data def convert_df(df): return df.to_csv(index=False).encode('utf-8')  csv = convert_df(df) st.download_button( "Press to Download",   csv, "file.csv", "text/csv",   key='download-csv' ) `
