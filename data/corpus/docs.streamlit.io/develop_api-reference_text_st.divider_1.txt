Summary: The st.divider function in Streamlit (version 1.44.0) is used to display a horizontal rule. It can also be achieved with st.write("---") or simply by writing "---" in the script.

## [](https://docs.streamlit.io/develop/api-reference/text/st.divider#stdivider)st.divider
Streamlit VersionVersion 1.44.0Version 1.43.0Version 1.42.0Version 1.41.0Version 1.40.0Version 1.39.0Version 1.38.0Version 1.37.0Version 1.36.0Version 1.35.0Version 1.34.0Version 1.33.0Version 1.32.0Version 1.31.0Version 1.30.0Version 1.29.0Version 1.28.0Version 1.27.0Version 1.26.0Version 1.25.0Version 1.24.0Version 1.23.0Version 1.22.0Version 1.21.0Version 1.20.0
Display a horizontal rule.
Note
You can achieve the same effect with st.write("---") or even just "---" in your script (via magic).
Function signature[[source]](https://github.com/streamlit/streamlit/blob/1.44.0/lib/streamlit/elements/markdown.py#L261 "View st.divider source code on GitHub")  
---  
st.divider()  
#### Example
```
import streamlit as st
st.divider()
```
