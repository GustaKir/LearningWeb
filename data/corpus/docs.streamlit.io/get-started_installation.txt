# [](https://docs.streamlit.io/get-started/installation#install-streamlit)Install Streamlit
There are multiple ways to set up your development environment and install Streamlit. Read below to understand these options. Developing locally with Python installed on your own computer is the most common scenario.
## [](https://docs.streamlit.io/get-started/installation#summary-for-experts)Summary for experts
  1. Set up your Python development environment.
  2. Run: 
`pip install streamlit `
  3. Validate the installation by running our Hello app: 
`streamlit hello `
  4. Jump to our [Basic concepts](https://docs.streamlit.io/get-started/fundamentals/main-concepts).
