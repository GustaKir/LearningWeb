Title: Streamlit Camera Input with PyTorch Integration
URL: https://docs.streamlit.io/develop/api-reference/widgets/st.camera_input
Summary: This documentation chunk explains how to use the Streamlit `camera_input` widget to capture images. It provides a code example for reading the image file buffer as a 3D uint8 tensor using PyTorch, demonstrating how to convert image data and check the resulting tensor's type and shape.
---

ou have installed [PyTorch](https://pytorch.org/) and [NumPy](https://numpy.org/).
To read the image file buffer as a 3 dimensional uint8 tensor with PyTorch:
`import streamlit as st import torch import numpy as np img_file_buffer = st.camera_input("Take a picture") if img_file_buffer is not None: # To read image file buffer as a 3D uint8 tensor with PyTorch:   bytes_data = img_file_buffer.getvalue()   torch_img = torch.ops.image.decode_image(     torch.from_numpy(np.frombuffer(bytes_data, np.uint8)), 3 ) # Check the type of torch_img: # Should output: <class 'torch.Tensor'>   st.write(type(torch_img)) # Check the shape of torch_img: # Should output shape: torch.Size([channels, height, width])   st.write(torch_img.shape) `
