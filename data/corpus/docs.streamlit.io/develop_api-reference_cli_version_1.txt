Title: Streamlit Version Command
URL: https://docs.streamlit.io/develop/api-reference/cli/version
Summary: The `$ streamlit version` command prints the version number of Streamlit, functioning the same as `streamlit --version`. This section provides the syntax for using the command.
---

## [](https://docs.streamlit.io/develop/api-reference/cli/version#-streamlit-version)`$ streamlit version`
Print Streamlit's version number. This command is equivalent to executing `streamlit --version`.
### [](https://docs.streamlit.io/develop/api-reference/cli/version#syntax)Syntax
`streamlit version `
