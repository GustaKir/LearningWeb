Title: st.get_option Documentation
URL: https://docs.streamlit.io/develop/api-reference/configuration/st.get_option
Summary: The `st.get_option` function in Streamlit returns the current value of a specified configuration option, identified by a key in the format 'section.optionName'. Users can view all available options by running 'streamlit config show' in their terminal. The documentation includes the function signature and provides an example of its usage.

Return the current value of a given Streamlit configuration option.
Run `streamlit config show` in a terminal to see all available options.
Function signature[[source]](https://github.com/streamlit/streamlit/blob/1.44.0/lib/streamlit/config.py#L169 "View st.get_option source code on GitHub")  
---  
st.get_option(key)  
Parameters  
key (str) |  The config option key of the form "section.optionName". To see all available options, run `streamlit config show` in a terminal.  
#### Example
```
import streamlit as st
color = st.get_option("theme.primaryColor")
```
