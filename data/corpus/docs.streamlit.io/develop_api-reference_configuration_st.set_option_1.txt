Title: st.set_option: Setting Configuration Options in Streamlit
URL: https://docs.streamlit.io/develop/api-reference/configuration/st.set_option
Summary: The `st.set_option` function allows users to set specific client configuration options in Streamlit scripts, including error details display, sidebar navigation, and toolbar mode. Attempting to set other options will result in a `StreamlitAPIException`. Users may need to trigger a rerun of the app to observe effects from configuration changes.
---

## [](https://docs.streamlit.io/develop/api-reference/configuration/st.set_option#stset_option)st.set_option

Set a configuration option.
Currently, only `client` configuration options can be set within the script itself:
>   * `client.showErrorDetails`
>   * `client.showSidebarNavigation`
>   * `client.toolbarMode`
> 

Calling `st.set_option` with any other option will raise a `StreamlitAPIException`. When changing a configuration option in a running app, you may need to trigger a rerun after changing the option to see the effects.
Run `streamlit config show` in a terminal to see all available options.
Function signature[[source]](https://github.com/streamlit/streamlit/blob/1.44.0/lib/streamlit/config.py#L121 "View st.set_option source code on GitHub")  
---  
st.set_option(key, value)  
Parameters  
key (str) |  The config option key of the form "section.optionName". To see all available options, run `streamlit config show` in a terminal.  
value (null) |  The new value to assign to this config option.  
#### Example
```
import streamlit as st
st.set_option("client.showErrorDetails", True)
```
