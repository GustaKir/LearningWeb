Title: st.switch_page Function in Streamlit
URL: https://docs.streamlit.io/develop/api-reference/navigation/st.switch_page
Summary: The `st.switch_page` function allows users to programmatically switch the current page in a Streamlit multipage application. When invoked, it halts the execution of the current page and runs the specified page as if it were selected through the sidebar. The target page must be compatible with Streamlit's multipage structure and cannot be an arbitrary Python script.

Programmatically switch the current page in a multipage app.
When `st.switch_page` is called, the current page execution stops and the specified page runs as if the user clicked on it in the sidebar navigation. The specified page must be recognized by Streamlit's multipage architecture (your main Python file or a Python file in a `pages/` folder). Arbitrary Python scripts cannot be passed to `st.switch_page`.
Function signature[[source]](https://github.com/streamlit/streamlit/blob/1.44.0/lib/streamlit/commands/execution_control.py#L156 "View st.switch_page source code on GitHub")  
---  
st.switch_page(page)  
Parameters  
page (str, Path, or st.Page) |  The file path (relative to the main script) or an st.Page indicating the page to switch to.
