# [](https://docs.streamlit.io/develop/quick-reference/release-notes/2020#2020-release-notes)2020 release notes
This page contains release notes for Streamlit versions released in 2020. For the latest version of Streamlit, see [Release notes](https://docs.streamlit.io/develop/quick-reference/release-notes).

_Release date: December 17, 2020_
**Notable Changes**
  * üêç Streamlit can now be installed on Python 3.9. Streamlit components are not yet compatible with Python 3.9 and must use version 3.8 or earlier.
  * üß± Streamlit Components now allows same origin, enabling features provided by the browser such as a webcam component.
  * üêô Fix Streamlit sharing deploy experience for users running on Git versions 2.7.0 or earlier.
  * üß∞ Handle unexpected closing of uploaded files for [`st.file_uploader`](https://docs.streamlit.io/en/0.72.0/api.html#streamlit.file_uploader).
