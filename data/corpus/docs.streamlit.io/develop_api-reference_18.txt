Title: Streamlit API Reference Overview
URL: https://docs.streamlit.io/develop/api-reference
Summary: This document provides an overview of various Streamlit components used for app testing, including `st.button`, `st.chat_input`, `st.checkbox`, and `st.color_picker`. It describes how to interact with these elements programmatically, such as asserting values and invoking actions like clicks and selections.
