Title: Streamlit Release Notes - 2023
URL: https://docs.streamlit.io/develop/quick-reference/release-notes/2023
Summary: This release introduces notable updates including support for Pandas 2.0, customization options for toolbar visibility using the `client.toolbarMode` configuration, and a change to log management by placing Streamlit logs in the 'streamlit' namespace. Additionally, CLI parameters are restricted from setting sensitive configuration values.
