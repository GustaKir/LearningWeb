Title: Streamlit Checkbox API Reference
URL: https://docs.streamlit.io/develop/api-reference/widgets/st.checkbox
Summary: This section provides an example of using the checkbox widget in Streamlit. The code snippet demonstrates creating a checkbox labeled 'I agree', and displays a confirmation message when the checkbox is checked. Additionally, it includes links to related videos and navigation to previous and next documentation sections.
---

```
import streamlit as st
agree = st.checkbox("I agree")
if agree:
  st.write("Great!")
```
