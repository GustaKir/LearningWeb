Title: AppTest.sidebar Overview
URL: https://docs.streamlit.io/develop/api-reference/app-testing/st.testing.v1.apptest
Summary: The AppTest.sidebar function provides a sequence of all elements within the Streamlit sidebar, allowing users to query elements similar to the AppTest class. This includes accessing elements like checkboxes through a Block container.
---

For example, `at.selectbox[0]` for the first widget or `at.selectbox(key="my_key")` for a widget with a given key.
