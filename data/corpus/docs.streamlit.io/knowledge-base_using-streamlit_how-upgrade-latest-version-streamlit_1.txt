Title: Upgrading Streamlit Using Conda and Poetry
URL: https://docs.streamlit.io/knowledge-base/using-streamlit/how-upgrade-latest-version-streamlit
Summary: This document provides instructions for upgrading Streamlit to the latest version using Conda and Poetry. It includes the command for updating Streamlit with Conda and a corresponding command for Poetry, ensuring users can verify they have the latest version.
---

`conda update -c conda-forge streamlit -y streamlit version `
## [](https://docs.streamlit.io/knowledge-base/using-streamlit/how-upgrade-latest-version-streamlit#poetry)Poetry
In order to get the latest version of Streamlit with [Poetry](https://python-poetry.org/) and verify you have the latest version, run:
`poetry update streamlit streamlit version `
