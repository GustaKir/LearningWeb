Title: Streamlit CLI Help Command
URL: https://docs.streamlit.io/develop/api-reference/cli/help
Summary: $ streamlit help is a command that displays available commands for the Streamlit CLI tool, serving the same purpose as `streamlit --help`.
---

## [](https://docs.streamlit.io/develop/api-reference/cli/help#-streamlit-help)`$ streamlit help`
Print the available commands for the Streamlit CLI tool. This command is equivalent to executing `streamlit --help`.
### [](https://docs.streamlit.io/develop/api-reference/cli/help#syntax)Syntax
`streamlit help `
