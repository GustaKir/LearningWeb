Title: Release Notes for Streamlit Version 1.23.0
URL: https://docs.streamlit.io/develop/quick-reference/release-notes/2023
Summary: Released on June 1, 2023, Streamlit 1.23.0 introduces the st.data_editor widget for editing DataFrames and other data in a table-like format. It includes a breaking change in how edited data is represented in st.session_state. Additionally, the new Column configuration API offers methods for customizing the display and behavior of data columns.
---

## [](https://docs.streamlit.io/develop/quick-reference/release-notes/2023#version-1230)**Version 1.23.0**
_Release date: June 1, 2023_
**Highlights**
  * ‚úÇÔ∏è Announcing the general availability of [st.data_editor](https://docs.streamlit.io/develop/api-reference/data/st.data_editor), a widget that allows you to edit DataFrames and many other data structures in a table-like UI. **Breaking change:** the data editor's representation used in `st.session_state` was altered. Find out more about the new format in [Access edited data](https://docs.streamlit.io/develop/concepts/design/dataframes#access-edited-data).
  * ‚öôÔ∏è Introducing the [Column configuration API](https://docs.streamlit.io/develop/api-reference/data/st.column_config) with a suite of methods to configure the display and editing behavior of `st.dataframe` and `st.data_editor` columns (e.g. their title, visibility, type, or format). Keep an eye out for a detailed [blog post](https://blog.streamlit.io/) and in-depth [documentation](https://docs.streamlit.io/develop/concepts/design/dataframes#configuring-columns) upcoming in the next two weeks.
  * üîå Learn to use `st.experimental_connection` to create and manage data connections in your apps with the new [Connecting to data](https://docs.streamlit.io/develop/concepts/connections/connecting-to-data) docs and [video tutorial](https://www.youtube.com/watch?v=xQwDfW7UHMo).


**Notable Changes**
  * üìä Streamlit now supports Protobuf 4 and Altair 5 ([#6215](https://github.com/streamlit/streamlit/issues/6215), [#6618](https://github.com/streamlit/streamlit/pull/6618), [#5626](https://github.com/streamlit/streamlit/issues/5626), [#6622](https://github.com/streamlit/streamlit/pull/6622)).
  * ‚òéÔ∏è st.dataframe and st.data_editor can hide index columns with `hide_index`, specify the display order of columns with `column_order`, and disable editing for individual columns with the `disabled` parameter.
  * ‚è±Ô∏è The `ttl` parameter in [st.cache_data](https://docs.streamlit.io/develop/api-reference/caching-and-state/st.cache_data) and [st.cache_resource](https://docs.streamlit.io/develop/api-reference/caching-and-state/st.cache_resource) accepts formatted strings, so you can simply say `ttl="30d"`, `ttl="1h30m"` and any other combination of `w`, `d`, `h`, `m`, `s` supported by [Pandas's Timedelta constructor](https://pandas.pydata.org/docs/reference/api/pandas.Timedelta.html) ([#6560](https://github.com/streamlit/streamlit/pull/6560)).
  * üìÇ `st.file_uploader` now interprets the `type` parameter more accurately. For example, "jpg" or ".jpg" now accept both "jpg" and "jpeg" extensions. This functionality has also been extended to "mpeg/mpg", "tiff/tif", "html/htm", and "mpeg4/mp4".
  * ü§´ The new `global.disableWidgetStateDuplicationWarning` configuration option allows the silencing of warnings triggered by setting widget default values and keyed session state values concurrently ([#3605](https://github.com/streamlit/streamlit/issues/3605), [#6640](https://github.com/streamlit/streamlit/pull/6640)).
