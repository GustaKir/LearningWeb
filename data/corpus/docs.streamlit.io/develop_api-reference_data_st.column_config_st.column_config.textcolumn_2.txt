Title: Streamlit TextColumn Configuration
URL: https://docs.streamlit.io/develop/api-reference/data/st.column_config/st.column_config.textcolumn
Summary: This documentation chunk illustrates how to use the TextColumn feature in Streamlit's data editor. It provides an example of setting up a data frame with a configurable text column, including properties like help text, default value, maximum character limit, and validation regex for user input.
---

```
import pandas as pd
import streamlit as st
data_df = pd.DataFrame(
  {
    "widgets": ["st.selectbox", "st.number_input", "st.text_area", "st.button"],
  }
)
st.data_editor(
  data_df,
  column_config={
    "widgets": st.column_config.TextColumn(
      "Widgets",
      help="Streamlit **widget** commands ðŸŽˆ",
      default="st.",
      max_chars=50,
      validate=r"^st\.[a-z_]+$",
    )
  },
  hide_index=True,
)
```
