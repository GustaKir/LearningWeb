Title: Customizing Selectbox Widget in Streamlit
URL: https://docs.streamlit.io/develop/api-reference/widgets/st.selectbox
Summary: This section explains how to customize the label visibility of select widgets in Streamlit using the `label_visibility` parameter, which includes options for 'visible', 'hidden', and 'collapsed'. It also discusses how to disable select widgets with the `disabled` parameter, including an example using session state to manage widget states.

Select widgets can customize how to hide their labels with the `label_visibility` parameter. If "hidden", the label doesnâ€™t show but there is still empty space for it above the widget (equivalent to `label=""`). If "collapsed", both the label and the space are removed. Default is "visible". Select widgets can also be disabled with the `disabled` parameter:
`import streamlit as st # Store the initial value of widgets in session state if "visibility" not in st.session_state:   st.session_state.visibility = "visible"   st.session_state.disabled = False col1, col2 = st.columns(2) with col1:   st.checkbox("Disable selectbox widget", key="disabled")   st.radio( "Set selectbox label visibility ðŸ‘‰",     key="visibility",     options=["visible", "hidden", "collapsed"], ) with col2:   option = st.selectbox( "How would you like to be contacted?", ("Email", "Home phone", "Mobile phone"),     label_visibility=st.session_state.visibility,     disabled=st.session_state.disabled, ) `
