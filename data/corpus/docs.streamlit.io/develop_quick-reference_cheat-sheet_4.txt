Title: Streamlit Cheat Sheet: Caching and Progress Indicators
URL: https://docs.streamlit.io/develop/quick-reference/cheat-sheet
Summary: This section outlines the usage of 'st.cache_resource' for caching functions in Streamlit, including how to create, retrieve, and clear cached items. It also covers displaying progress and status updates during processes, utilizing features like spinners, progress bars, and various notification messages for the user.
---

@st.cache_resource def foo(bar): # Create and return a non-data object return session # Executes foo s1 = foo(ref1) # Does not execute foo # Returns cached item by reference, s1 == s2 s2 = foo(ref1) # Different arg, so function foo executes s3 = foo(ref2) # Clear the cached value for foo(ref1) foo.clear(ref1) # Clear all cached entries for this function foo.clear() # Clear all global resources from cache st.cache_resource.clear() `
