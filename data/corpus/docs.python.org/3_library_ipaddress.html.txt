Title: ipaddress — IPv4/IPv6 manipulation library
URL: https://docs.python.org/3/library/ipaddress.html
Summary: The `ipaddress` module in Python provides tools for creating, manipulating, and operating on IPv4 and IPv6 addresses and networks. It includes convenience factory functions, address objects, and methods for converting addresses to strings and integers.
