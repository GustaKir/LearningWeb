Title: Configuring Python Build Environment Variables
URL: https://docs.python.org/3/using/configure.html
Summary: This section discusses important environment variables for configuring the Python build process, including `LDFLAGS` for specifying linker flags, `LIBS` for passing libraries to the linker, `LDSHARED` for defining the command to build shared libraries, and `BLDSHARED` for additional shared library build commands.
---

LDFLAGS[¶](https://docs.python.org/3/using/configure.html#envvar-LDFLAGS "Link to this definition")
    
Linker flags, e.g. `-L_lib_dir_`if you have libraries in a nonstandard directory _lib_dir_.
Both [`CPPFLAGS`](https://docs.python.org/3/using/configure.html#envvar-CPPFLAGS) and [`LDFLAGS`](https://docs.python.org/3/using/configure.html#envvar-LDFLAGS) need to contain the shell’s value to be able to build extension modules using the directories specified in the environment variables. 

LIBS[¶](https://docs.python.org/3/using/configure.html#envvar-LIBS "Link to this definition")
    
Linker flags to pass libraries to the linker when linking the Python executable.
Example: `-lrt`. 

LDSHARED[¶](https://docs.python.org/3/using/configure.html#envvar-LDSHARED "Link to this definition")
    
Command to build a shared library.
Default: `@LDSHARED@ $(PY_LDFLAGS)`. 

BLDSHARED[¶](https://docs.python.org/3/using/configure.html#envvar-BLDSHARED "Link to this definition")
    
Command to build `libpython` shared library.
Default: `@BLDSHARED@ $(PY_CORE_LDFLAGS)`. 

PY_LDFLAGS[¶](https://docs.python.org/3/using/configure.html#envvar-PY_LDFLAGS "Link to this definition")
    
Default: `$(CONFIGURE_LDFLAGS) $(LDFLAGS)`. 

PY_LDFLAGS_NODIST[¶](https://docs.python.org/3/using/configure.html#envvar-PY_LDFLAGS_NODIST "Link to this definition")
    
Default: `$(CONFIGURE_LDFLAGS_NODIST) $(LDFLAGS_NODIST)`.
Added in version 3.8. 

PY_CORE_LDFLAGS[¶](https://docs.python.org/3/using/configure.html#envvar-PY_CORE_LDFLAGS "Link to this definition")
    
Linker flags used for building the interpreter object files.
Added in version 3.8.
