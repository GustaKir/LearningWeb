Title: MIMEPart Class in Python's Email Library
URL: https://docs.python.org/3/library/email.message.html
Summary: The `MIMEPart` class represents a subpart of a MIME message, similar to `EmailMessage`, but without adding _MIME-Version_ headers when `set_content()` is called, since sub-parts do not require their own headers.
---

_class_ email.message.MIMEPart(_policy =default_)[¶](https://docs.python.org/3/library/email.message.html#email.message.MIMEPart "Link to this definition")
    
This class represents a subpart of a MIME message. It is identical to [`EmailMessage`](https://docs.python.org/3/library/email.message.html#email.message.EmailMessage "email.message.EmailMessage"), except that no _MIME-Version_ headers are added when [`set_content()`](https://docs.python.org/3/library/email.message.html#email.message.EmailMessage.set_content "email.message.EmailMessage.set_content") is called, since sub-parts do not need their own _MIME-Version_ headers.
