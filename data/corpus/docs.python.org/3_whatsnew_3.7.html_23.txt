Title: Enhancements in the Time Module (PEP 564)
URL: https://docs.python.org/3/whatsnew/3.7.html
Summary: PEP 564 introduces six new functions to the time module that provide nanosecond resolution: `clock_gettime_ns`, `clock_settime_ns`, `monotonic_ns`, `perf_counter_ns`, `process_time_ns`, and `time_ns`.

[**PEP 564**](https://peps.python.org/pep-0564/) adds six new functions with nanosecond resolution to the [`time`](https://docs.python.org/3/library/time.html#module-time "time: Time access and conversions.") module:
  * [`time.clock_gettime_ns()`](https://docs.python.org/3/library/time.html#time.clock_gettime_ns "time.clock_gettime_ns")
  * [`time.clock_settime_ns()`](https://docs.python.org/3/library/time.html#time.clock_settime_ns "time.clock_settime_ns")
  * [`time.monotonic_ns()`](https://docs.python.org/3/library/time.html#time.monotonic_ns "time.monotonic_ns")
  * [`time.perf_counter_ns()`](https://docs.python.org/3/library/time.html#time.perf_counter_ns "time.perf_counter_ns")
  * [`time.process_time_ns()`](https://docs.python.org/3/library/time.html#time.process_time_ns "time.process_time_ns")
  * [`time.time_ns()`](https://docs.python.org/3/library/time.html#time.time_ns "time.time_ns")


New clock identifiers have been added:
  * [`time.CLOCK_BOOTTIME`](https://docs.python.org/3/library/time.html#time.CLOCK_BOOTTIME "time.CLOCK_BOOTTIME") (Linux): Identical to [`time.CLOCK_MONOTONIC`](https://docs.python.org/3/library/time.html#time.CLOCK_MONOTONIC "time.CLOCK_MONOTONIC"), except it also includes any time that the system is suspended.
  * [`time.CLOCK_PROF`](https://docs.python.org/3/library/time.html#time.CLOCK_PROF "time.CLOCK_PROF") (FreeBSD, NetBSD and OpenBSD): High-resolution per-process CPU timer.
  * [`time.CLOCK_UPTIME`](https://docs.python.org/3/library/time.html#time.CLOCK_UPTIME "time.CLOCK_UPTIME") (FreeBSD, OpenBSD): Time whose absolute value is the time the system has been running and not suspended, providing accurate uptime measurement.
