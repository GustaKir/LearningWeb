Title: Enhancements to IDLE's ConfigDialog
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: The updates include several improvements to IDLE's ConfigDialog, such as reorganizing method structures for better testability, factoring out a VarTrace class for managing variable and callback pairs, adding extensive tests for the General tab, and enhancing the font configuration page with clearer documentation and simplified attribute names.
