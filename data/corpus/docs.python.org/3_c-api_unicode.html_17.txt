Title: Unicode-Escape String Encoding and Raw Unicode Escape Codecs
URL: https://docs.python.org/3/c-api/unicode.html
Summary: This documentation chunk describes the function `PyUnicode_AsUnicodeEscapeString`, which encodes a Unicode object into a bytes object using Unicode-Escape encoding. It mentions error handling as 'strict' and states that NULL is returned if any exception occurs. It also introduces the Raw-Unicode-Escape codec APIs.

### Latin-1 Codecs[¶](https://docs.python.org/3/c-api/unicode.html#latin-1-codecs "Link to this heading")
These are the Latin-1 codec APIs: Latin-1 corresponds to the first 256 Unicode ordinals and only these are accepted by the codecs during encoding.

### ASCII Codecs[¶](https://docs.python.org/3/c-api/unicode.html#ascii-codecs "Link to this heading")
These are the ASCII codec APIs. Only 7-bit ASCII data is accepted. All other codes generate errors.
