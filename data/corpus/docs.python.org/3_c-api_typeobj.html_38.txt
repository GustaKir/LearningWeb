## Mapping Object Structures[¶](https://docs.python.org/3/c-api/typeobj.html#mapping-object-structures "Link to this heading") 

typePyMappingMethods[¶](https://docs.python.org/3/c-api/typeobj.html#c.PyMappingMethods "Link to this definition")
    
This structure holds pointers to the functions which an object uses to implement the mapping protocol. It has three members: 

[lenfunc](https://docs.python.org/3/c-api/typeobj.html#c.lenfunc "lenfunc")[PyMappingMethods](https://docs.python.org/3/c-api/typeobj.html#c.PyMappingMethods "PyMappingMethods").mp_length[¶](https://docs.python.org/3/c-api/typeobj.html#c.PyMappingMethods.mp_length "Link to this definition")
    
This function is used by [`PyMapping_Size()`](https://docs.python.org/3/c-api/mapping.html#c.PyMapping_Size "PyMapping_Size") and [`PyObject_Size()`](https://docs.python.org/3/c-api/object.html#c.PyObject_Size "PyObject_Size"), and has the same signature. This slot may be set to `NULL` if the object has no defined length. 

[binaryfunc](https://docs.python.org/3/c-api/typeobj.html#c.binaryfunc "binaryfunc")[PyMappingMethods](https://docs.python.org/3/c-api/typeobj.html#c.PyMappingMethods "PyMappingMethods").mp_subscript[¶](https://docs.python.org/3/c-api/typeobj.html#c.PyMappingMethods.mp_subscript "Link to this definition")
    
This function is used by [`PyObject_GetItem()`](https://docs.python.org/3/c-api/object.html#c.PyObject_GetItem "PyObject_GetItem") and [`PySequence_GetSlice()`](https://docs.python.org/3/c-api/sequence.html#c.PySequence_GetSlice "PySequence_GetSlice"), and has the same signature as `PyObject_GetItem()`. This slot must be filled for the [`PyMapping_Check()`](https://docs.python.org/3/c-api/mapping.html#c.PyMapping_Check "PyMapping_Check") function to return `1`, it can be `NULL` otherwise.
