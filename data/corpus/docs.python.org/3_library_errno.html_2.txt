Title: Error Codes in Python's errno Module
URL: https://docs.python.org/3/library/errno.html
Summary: This section discusses various error codes in the Python errno module, including EWOULDBLOCK, which indicates an operation would block and is associated with BlockingIOError, as well as ENOMSG, EIDRM, ECHRNG, EL2NSYNC, EL3HLT, and EL3RST, each representing different error statuses.
---

errno.EWOULDBLOCK[¶](https://docs.python.org/3/library/errno.html#errno.EWOULDBLOCK "Link to this definition")
    
Operation would block. This error is mapped to the exception [`BlockingIOError`](https://docs.python.org/3/library/exceptions.html#BlockingIOError "BlockingIOError").
