Title: Email Package Documentation
URL: https://docs.python.org/3/library/email.html
Summary: This section outlines the documentation for the Python email package, highlighting its relevance for applications using the compat32 API for backward compatibility. It notes changes introduced in version 3.6, including a reorganization to emphasize the new EmailMessage and EmailPolicy APIs, and provides links to various components of the email package.
---

This documentation is also relevant for applications that are still using the [`compat32`](https://docs.python.org/3/library/email.policy.html#email.policy.compat32 "email.policy.compat32") API for backward compatibility reasons.
Changed in version 3.6: Docs reorganized and rewritten to promote the new [`EmailMessage`](https://docs.python.org/3/library/email.message.html#email.message.EmailMessage "email.message.EmailMessage")/[`EmailPolicy`](https://docs.python.org/3/library/email.policy.html#email.policy.EmailPolicy "email.policy.EmailPolicy") API.
