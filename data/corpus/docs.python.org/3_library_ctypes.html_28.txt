Title: ctypes._Pointer.contents Documentation
URL: https://docs.python.org/3/library/ctypes.html
Summary: The 'contents' attribute of a ctypes pointer returns the object that the pointer references. Assigning a new value to this attribute updates the pointer to point to the newly assigned object.
---

contents[¶](https://docs.python.org/3/library/ctypes.html#ctypes._Pointer.contents "Link to this definition")
    
Returns the object to which to pointer points. Assigning to this attribute changes the pointer to point to the assigned object.
