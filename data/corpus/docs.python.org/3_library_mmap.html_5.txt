Title: mmap Advice Constants
URL: https://docs.python.org/3/library/mmap.html
Summary: This section lists various advice constants for memory mapping including MADV_AUTOSYNC, MADV_NOCORE, MADV_CORE, MADV_PROTECT, MADV_FREE_REUSABLE, and MADV_FREE_REUSE, which can be used with the mmap.madvise() function. Availability may vary across systems, and these constants were introduced in Python version 3.8.

mmap.MADV_FREE_REUSE[¶](https://docs.python.org/3/library/mmap.html#mmap.MADV_FREE_REUSE "Link to this definition")
    
These options can be passed to [`mmap.madvise()`](https://docs.python.org/3/library/mmap.html#mmap.mmap.madvise "mmap.mmap.madvise"). Not every option will be present on every system.
Availability: Systems with the madvise() system call.
Added in version 3.8.
