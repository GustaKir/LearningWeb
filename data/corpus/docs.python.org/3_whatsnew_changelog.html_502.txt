Title: Documentation Changes in Python
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: This section outlines recent updates to Python documentation, including the removal of obsolete comments, documentation of relative imports, and the addition of a new translation in Simplified Chinese. It also notes improvements to docstrings and code duplication avoidance in examples.
---

### Documentation[¶](https://docs.python.org/3/whatsnew/changelog.html#id449 "Link to this heading")
  * [bpo-36625](https://bugs.python.org/issue?@action=redirect&bpo=36625): Remove obsolete comments from docstrings in fractions.Fraction
  * [bpo-30840](https://bugs.python.org/issue?@action=redirect&bpo=30840): Document relative imports
  * [bpo-36523](https://bugs.python.org/issue?@action=redirect&bpo=36523): Add docstring for io.IOBase.writelines().
  * [bpo-36425](https://bugs.python.org/issue?@action=redirect&bpo=36425): New documentation translation: [Simplified Chinese](https://docs.python.org/zh-cn/).
  * [bpo-36345](https://bugs.python.org/issue?@action=redirect&bpo=36345): Avoid the duplication of code from `Tools/scripts/serve.py` in using the `literalinclude` directive for the basic wsgiref-based web server in the documentation of [`wsgiref`](https://docs.python.org/3/library/wsgiref.html#module-wsgiref "wsgiref: WSGI Utilities and Reference Implementation."). Contributed by Stéphane Wirtel.
  * [bpo-36345](https://bugs.python.org/issue?@action=redirect&bpo=36345): Using the code of the `Tools/scripts/serve.py` script as an example in the [`wsgiref`](https://docs.python.org/3/library/wsgiref.html#module-wsgiref "wsgiref: WSGI Utilities and Reference Implementation.") documentation. Contributed by Stéphane Wirtel.
  * [bpo-36157](https://bugs.python.org/issue?@action=redirect&bpo=36157): Added documentation for PyInterpreterState_Main().
  * [bpo-33043](https://bugs.python.org/issue?@action=redirect&bpo=33043): Updates the docs.python.org page with the addition of a ‘Contributing to Docs’ link at the end of the page (between ‘Reporting Bugs’ and ‘About Documentation’). Updates the ‘Found a Bug’ page with additional links and information in the Documentation Bugs section.
  * [bpo-35581](https://bugs.python.org/issue?@action=redirect&bpo=35581): @typing.type_check_only now allows type stubs to mark functions and classes not available during runtime.
  * [bpo-33832](https://bugs.python.org/issue?@action=redirect&bpo=33832): Add glossary entry for ‘magic method’.
  * [bpo-32913](https://bugs.python.org/issue?@action=redirect&bpo=32913): Added re.Match.groupdict example to regex HOWTO.
