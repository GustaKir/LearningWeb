Title: Python 3.13.0 Release Notes
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: The release notes for Python 3.13.0 and its release candidate include important bug fixes in core and builtins. Notably, issues with `tokenize.untokenize()` and a compiler bug related to reachability analysis have been addressed. The final version was released on October 7, 2024, following the third release candidate from October 1, 2024.
---

## Python 3.13.0 final[¶](https://docs.python.org/3/whatsnew/changelog.html#python-3-13-0-final "Link to this heading")
_Release date: 2024-10-07_
### Core and Builtins[¶](https://docs.python.org/3/whatsnew/changelog.html#id28 "Link to this heading")
  * [gh-125008](https://github.com/python/cpython/issues/125008): Fix [`tokenize.untokenize()`](https://docs.python.org/3/library/tokenize.html#tokenize.untokenize "tokenize.untokenize") producing invalid syntax for double braces preceded by certain escape characters.
  * [gh-124871](https://github.com/python/cpython/issues/124871): Fix compiler bug (in some versions of 3.13) where an assertion fails during reachability analysis.


## Python 3.13.0 release candidate 3[¶](https://docs.python.org/3/whatsnew/changelog.html#python-3-13-0-release-candidate-3 "Link to this heading")
_Release date: 2024-10-01_
### macOS[¶](https://docs.python.org/3/whatsnew/changelog.html#id29 "Link to this heading")
  * [gh-123797](https://github.com/python/cpython/issues/123797): Check for runtime availability of `ptsname_r` function on macos.


### Windows[¶](https://docs.python.org/3/whatsnew/changelog.html#id30 "Link to this heading")
  * [gh-124609](https://github.com/python/cpython/issues/124609): Fix `_Py_ThreadId` for Windows builds using MinGW. Patch by Tony Roberts.
  * [gh-124254](https://github.com/python/cpython/issues/124254): Ensures experimental free-threaded binaries remain installed when updating.
  * [gh-123915](https://github.com/python/cpython/issues/123915): Ensure that `Tools\msi\buildrelease.bat` uses different directories for AMD64 and ARM64 builds.


### Tests[¶](https://docs.python.org/3/whatsnew/changelog.html#id31 "Link to this heading")
  * [gh-124378](https://github.com/python/cpython/issues/124378): Updated `test_ttk` to pass with Tcl/Tk 8.6.15.
