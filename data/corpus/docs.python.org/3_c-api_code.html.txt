# Code Objects[¶](https://docs.python.org/3/c-api/code.html#code-objects "Link to this heading")
Code objects are a low-level detail of the CPython implementation. Each one represents a chunk of executable code that hasn’t yet been bound into a function. 

typePyCodeObject[¶](https://docs.python.org/3/c-api/code.html#c.PyCodeObject "Link to this definition")
    
The C structure of the objects used to describe code objects. The fields of this type are subject to change at any time. 

[PyTypeObject](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject")PyCode_Type[¶](https://docs.python.org/3/c-api/code.html#c.PyCode_Type "Link to this definition")
    
This is an instance of [`PyTypeObject`](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject") representing the Python [code object](https://docs.python.org/3/reference/datamodel.html#code-objects). 

intPyCode_Check([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*co)[¶](https://docs.python.org/3/c-api/code.html#c.PyCode_Check "Link to this definition")
    
Return true if _co_ is a [code object](https://docs.python.org/3/reference/datamodel.html#code-objects). This function always succeeds. 

[Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")PyCode_GetNumFree([PyCodeObject](https://docs.python.org/3/c-api/code.html#c.PyCodeObject "PyCodeObject")*co)[¶](https://docs.python.org/3/c-api/code.html#c.PyCode_GetNumFree "Link to this definition")
    
Return the number of [free (closure) variables](https://docs.python.org/3/glossary.html#term-closure-variable) in a code object. 

intPyUnstable_Code_GetFirstFree([PyCodeObject](https://docs.python.org/3/c-api/code.html#c.PyCodeObject "PyCodeObject")*co)[¶](https://docs.python.org/3/c-api/code.html#c.PyUnstable_Code_GetFirstFree "Link to this definition")
    
_This is[ Unstable API](https://docs.python.org/3/c-api/stable.html#unstable-c-api). It may change without warning in minor releases._
Return the position of the first [free (closure) variable](https://docs.python.org/3/glossary.html#term-closure-variable) in a code object.
Changed in version 3.13: Renamed from `PyCode_GetFirstFree` as part of [Unstable C API](https://docs.python.org/3/c-api/stable.html#unstable-c-api). The old name is deprecated, but will remain available until the signature changes again.
