Title: CPython-Specific Implementation Details
URL: https://docs.python.org/3/library/sys.html
Summary: This section details the CPython-specific method for accessing command line options via '-X', noting that other Python implementations may handle these options differently. It references the addition of this feature in version 3.2.
---

**CPython implementation detail:** This is a CPython-specific way of accessing options passed through [`-X`](https://docs.python.org/3/using/cmdline.html#cmdoption-X). Other implementations may export them through other means, or not at all.
Added in version 3.2.
