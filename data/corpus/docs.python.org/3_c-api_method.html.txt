# Instance Method Objects[¶](https://docs.python.org/3/c-api/method.html#instance-method-objects "Link to this heading")
An instance method is a wrapper for a [`PyCFunction`](https://docs.python.org/3/c-api/structures.html#c.PyCFunction "PyCFunction") and the new way to bind a [`PyCFunction`](https://docs.python.org/3/c-api/structures.html#c.PyCFunction "PyCFunction") to a class object. It replaces the former call `PyMethod_New(func, NULL, class)`. 

[PyTypeObject](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject")PyInstanceMethod_Type[¶](https://docs.python.org/3/c-api/method.html#c.PyInstanceMethod_Type "Link to this definition")
    
This instance of [`PyTypeObject`](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject") represents the Python instance method type. It is not exposed to Python programs. 

intPyInstanceMethod_Check([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*o)[¶](https://docs.python.org/3/c-api/method.html#c.PyInstanceMethod_Check "Link to this definition")
    
Return true if _o_ is an instance method object (has type [`PyInstanceMethod_Type`](https://docs.python.org/3/c-api/method.html#c.PyInstanceMethod_Type "PyInstanceMethod_Type")). The parameter must not be `NULL`. This function always succeeds. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyInstanceMethod_New([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*func)[¶](https://docs.python.org/3/c-api/method.html#c.PyInstanceMethod_New "Link to this definition")
    _Return value: New reference._
Return a new instance method object, with _func_ being any callable object. _func_ is the function that will be called when the instance method is called. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyInstanceMethod_Function([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*im)[¶](https://docs.python.org/3/c-api/method.html#c.PyInstanceMethod_Function "Link to this definition")
    _Return value: Borrowed reference._
Return the function object associated with the instance method _im_.
