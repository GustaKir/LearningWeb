### 3.3.1. General Options[¶](https://docs.python.org/3/using/configure.html#general-options "Link to this heading") 

--enable-loadable-sqlite-extensions[¶](https://docs.python.org/3/using/configure.html#cmdoption-enable-loadable-sqlite-extensions "Link to this definition")
    
Support loadable extensions in the `_sqlite` extension module (default is no) of the [`sqlite3`](https://docs.python.org/3/library/sqlite3.html#module-sqlite3 "sqlite3: A DB-API 2.0 implementation using SQLite 3.x.") module.
See the [`sqlite3.Connection.enable_load_extension()`](https://docs.python.org/3/library/sqlite3.html#sqlite3.Connection.enable_load_extension "sqlite3.Connection.enable_load_extension") method of the [`sqlite3`](https://docs.python.org/3/library/sqlite3.html#module-sqlite3 "sqlite3: A DB-API 2.0 implementation using SQLite 3.x.") module.
Added in version 3.6. 

--disable-ipv6[¶](https://docs.python.org/3/using/configure.html#cmdoption-disable-ipv6 "Link to this definition")
    
Disable IPv6 support (enabled by default if supported), see the [`socket`](https://docs.python.org/3/library/socket.html#module-socket "socket: Low-level networking interface.") module. 

--enable-big-digits=[15|30][¶](https://docs.python.org/3/using/configure.html#cmdoption-enable-big-digits "Link to this definition")
    
Define the size in bits of Python [`int`](https://docs.python.org/3/library/functions.html#int "int") digits: 15 or 30 bits.
By default, the digit size is 30.
Define the `PYLONG_BITS_IN_DIGIT` to `15` or `30`.
See [`sys.int_info.bits_per_digit`](https://docs.python.org/3/library/sys.html#sys.int_info "sys.int_info"). 

--with-suffix=SUFFIX[¶](https://docs.python.org/3/using/configure.html#cmdoption-with-suffix "Link to this definition")
    
Set the Python executable suffix to _SUFFIX_.
The default suffix is `.exe` on Windows and macOS (`python.exe` executable), `.js` on Emscripten node, `.html` on Emscripten browser, `.wasm` on WASI, and an empty string on other platforms (`python` executable).
Changed in version 3.11: The default suffix on WASM platform is one of `.js`, `.html` or `.wasm`. 

--with-tzpath=<list of absolute paths separated by pathsep>[¶](https://docs.python.org/3/using/configure.html#cmdoption-with-tzpath "Link to this definition")
    
Select the default time zone search path for [`zoneinfo.TZPATH`](https://docs.python.org/3/library/zoneinfo.html#zoneinfo.TZPATH "zoneinfo.TZPATH"). See the [Compile-time configuration](https://docs.python.org/3/library/zoneinfo.html#zoneinfo-data-compile-time-config) of the [`zoneinfo`](https://docs.python.org/3/library/zoneinfo.html#module-zoneinfo "zoneinfo: IANA time zone support") module.
Default: `/usr/share/zoneinfo:/usr/lib/zoneinfo:/usr/share/lib/zoneinfo:/etc/zoneinfo`.
See [`os.pathsep`](https://docs.python.org/3/library/os.html#os.pathsep "os.pathsep") path separator.
Added in version 3.9. 

--without-decimal-contextvar[¶](https://docs.python.org/3/using/configure.html#cmdoption-without-decimal-contextvar "Link to this definition")
    
Build the `_decimal` extension module using a thread-local context rather than a coroutine-local context (default), see the [`decimal`](https://docs.python.org/3/library/decimal.html#module-decimal "decimal: Implementation of the General Decimal Arithmetic Specification.") module.
See [`decimal.HAVE_CONTEXTVAR`](https://docs.python.org/3/library/decimal.html#decimal.HAVE_CONTEXTVAR "decimal.HAVE_CONTEXTVAR") and the [`contextvars`](https://docs.python.org/3/library/contextvars.html#module-contextvars "contextvars: Context Variables") module.
Added in version 3.9. 

--with-dbmliborder=<list of backend names>[¶](https://docs.python.org/3/using/configure.html#cmdoption-with-dbmliborder "Link to this definition")
    
Override order to check db backends for the [`dbm`](https://docs.python.org/3/library/dbm.html#module-dbm "dbm: Interfaces to various Unix "database" formats.") module
A valid value is a colon (`:`) separated string with the backend names:
  * `ndbm`;
  * `gdbm`;
  * `bdb`.



--without-c-locale-coercion[¶](https://docs.python.org/3/using/configure.html#cmdoption-without-c-locale-coercion "Link to this definition")
    
Disable C locale coercion to a UTF-8 based locale (enabled by default).
Don’t define the `PY_COERCE_C_LOCALE` macro.
See [`PYTHONCOERCECLOCALE`](https://docs.python.org/3/using/cmdline.html#envvar-PYTHONCOERCECLOCALE) and the [**PEP 538**](https://peps.python.org/pep-0538/). 

--without-freelists[¶](https://docs.python.org/3/using/configure.html#cmdoption-without-freelists "Link to this definition")
    
Disable all freelists except the empty tuple singleton.
Added in version 3.11.
