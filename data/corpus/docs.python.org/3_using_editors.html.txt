# 8. Editors and IDEs[¶](https://docs.python.org/3/using/editors.html#editors-and-ides "Link to this heading")
There are a number of IDEs that support Python programming language. Many editors and IDEs provide syntax highlighting, debugging tools, and [**PEP 8**](https://peps.python.org/pep-0008/) checks.
## 8.1. IDLE — Python editor and shell[¶](https://docs.python.org/3/using/editors.html#idle-python-editor-and-shell "Link to this heading")
IDLE is Python’s Integrated Development and Learning Environment and is generally bundled with Python installs. If you are on Linux and do not have IDLE installed see [Installing IDLE on Linux](https://docs.python.org/3/using/unix.html#installing-idle-on-linux). For more information see the [IDLE docs](https://docs.python.org/3/library/idle.html#idle).
## 8.2. Other Editors and IDEs[¶](https://docs.python.org/3/using/editors.html#other-editors-and-ides "Link to this heading")
Python’s community wiki has information submitted by the community on Editors and IDEs. Please go to [Python Editors](https://wiki.python.org/moin/PythonEditors) and [Integrated Development Environments](https://wiki.python.org/moin/IntegratedDevelopmentEnvironments) for a comprehensive list.
