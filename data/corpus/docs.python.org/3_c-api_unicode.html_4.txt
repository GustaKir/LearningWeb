Title: Unicode C API Reference
URL: https://docs.python.org/3/c-api/unicode.html
Summary: This section provides descriptions of several C API functions related to Unicode characters in Python, including `Py_UNICODE_ISPRINTABLE`, which checks if a character is printable, and `Py_UNICODE_TOLOWER` and `Py_UNICODE_TOUPPER`, which convert characters to lower and upper case respectively.
---

intPy_UNICODE_ISPRINTABLE([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_ISPRINTABLE "Link to this definition")
    
Return `1` or `0` depending on whether _ch_ is a printable character, in the sense of [`str.isprintable()`](https://docs.python.org/3/library/stdtypes.html#str.isprintable "str.isprintable").
These APIs can be used for fast direct character conversions: 

[Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")Py_UNICODE_TOLOWER([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_TOLOWER "Link to this definition")
    
Return the character _ch_ converted to lower case. 

[Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")Py_UNICODE_TOUPPER([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_TOUPPER "Link to this definition")
    
Return the character _ch_ converted to upper case. 

[Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")Py_UNICODE_TOTITLE([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_TOTITLE "Link to this definition")
    
Return the character _ch_ converted to title case. 

intPy_UNICODE_TODECIMAL([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_TODECIMAL "Link to this definition")
    
Return the character _ch_ converted to a decimal positive integer. Return `-1` if this is not possible. This function does not raise exceptions. 

intPy_UNICODE_TODIGIT([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_TODIGIT "Link to this definition")
    
Return the character _ch_ converted to a single digit integer. Return `-1` if this is not possible. This function does not raise exceptions. 

doublePy_UNICODE_TONUMERIC([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_TONUMERIC "Link to this definition")
    
Return the character _ch_ converted to a double. Return `-1.0` if this is not possible. This function does not raise exceptions.
These APIs can be used to work with surrogates: 

intPy_UNICODE_IS_SURROGATE([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_IS_SURROGATE "Link to this definition")
    
Check if _ch_ is a surrogate (`0xD800 <= ch <= 0xDFFF`). 

intPy_UNICODE_IS_HIGH_SURROGATE([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_IS_HIGH_SURROGATE "Link to this definition")
    
Check if _ch_ is a high surrogate (`0xD800 <= ch <= 0xDBFF`). 

intPy_UNICODE_IS_LOW_SURROGATE([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")ch)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_IS_LOW_SURROGATE "Link to this definition")
    
Check if _ch_ is a low surrogate (`0xDC00 <= ch <= 0xDFFF`). 

[Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")Py_UNICODE_JOIN_SURROGATES([Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")high, [Py_UCS4](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4")low)[¶](https://docs.python.org/3/c-api/unicode.html#c.Py_UNICODE_JOIN_SURROGATES "Link to this definition")
    
Join two surrogate code points and return a single [`Py_UCS4`](https://docs.python.org/3/c-api/unicode.html#c.Py_UCS4 "Py_UCS4") value. _high_ and _low_ are respectively the leading and trailing surrogates in a surrogate pair. _high_ must be in the range [0xD800; 0xDBFF] and _low_ must be in the range [0xDC00; 0xDFFF].
