Title: Converting C Values to Python Data Structures
URL: https://docs.python.org/3/c-api/arg.html
Summary: This section describes how to convert sequences of C values into Python lists and dictionaries. A list is created with the same number of items, and a dictionary is formed by pairing consecutive C values as key-value pairs. Errors in the format string will trigger a SystemError, returning NULL. It also introduces the Py_VaBuildValue function, providing details about its return value.
---

`[items]` ([`list`](https://docs.python.org/3/library/stdtypes.html#list "list")) [_matching-items_]
    
Convert a sequence of C values to a Python list with the same number of items. 

`{items}` ([`dict`](https://docs.python.org/3/library/stdtypes.html#dict "dict")) [_matching-items_]
    
Convert a sequence of C values to a Python dictionary. Each pair of consecutive C values adds one item to the dictionary, serving as key and value, respectively.
If there is an error in the format string, the [`SystemError`](https://docs.python.org/3/library/exceptions.html#SystemError "SystemError") exception is set and `NULL` returned. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*Py_VaBuildValue(constchar*format, va_listvargs)[¶](https://docs.python.org/3/c-api/arg.html#c.Py_VaBuildValue "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Identical to [`Py_BuildValue()`](https://docs.python.org/3/c-api/arg.html#c.Py_BuildValue "Py_BuildValue"), except that it accepts a va_list rather than a variable number of arguments.
