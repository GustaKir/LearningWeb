Title: Changelog Overview for Python Updates
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: In this update, `ZipInfo` now issues a warning for duplicated entries instead of raising an exception. Notable changes include a fix for `pydoc`, the deprecation schedule for `check_home` in `sysconfig.is_python_build()`, and a resolution of extra references in `hashlib.sha256()` for free threading.
---

Reading duplicated entries (except the last one) by `ZipInfo` now emits a warning instead of raising an exception.
  * [gh-128772](https://github.com/python/cpython/issues/128772): Fix [`pydoc`](https://docs.python.org/3/library/pydoc.html#module-pydoc "pydoc: Documentation generator and online help system.") for methods with the `__module__` attribute equal to `None`.
  * [gh-92897](https://github.com/python/cpython/issues/92897): Scheduled the deprecation of the `check_home` argument of [`sysconfig.is_python_build()`](https://docs.python.org/3/library/sysconfig.html#sysconfig.is_python_build "sysconfig.is_python_build") to Python 3.15.
  * [gh-128657](https://github.com/python/cpython/issues/128657): Fix possible extra reference when using objects returned by [`hashlib.sha256()`](https://docs.python.org/3/library/hashlib.html#hashlib.sha256 "hashlib.sha256") under [free threading](https://docs.python.org/3/glossary.html#term-free-threading).
  * [gh-128703](https://github.com/python/cpython/issues/128703): Fix [`mimetypes.guess_type()`](https://docs.python.org/3/library/mimetypes.html#mimetypes.guess_type "mimetypes.guess_type") to use default mapping for empty `Content-Type` in registry.
  * [gh-128308](https://github.com/python/cpython/issues/128308): Support the _name_ keyword argument for eager tasks in [`asyncio.loop.create_task()`](https://docs.python.org/3/library/asyncio-eventloop.html#asyncio.loop.create_task "asyncio.loop.create_task"), [`asyncio.create_task()`](https://docs.python.org/3/library/asyncio-task.html#asyncio.create_task "asyncio.create_task") and [`asyncio.TaskGroup.create_task()`](https://docs.python.org/3/library/asyncio-task.html#asyncio.TaskGroup.create_task "asyncio.TaskGroup.create_task"), by passing on all _kwargs_ to the task factory set by [`asyncio.loop.set_task_factory()`](https://docs.python.org/3/library/asyncio-eventloop.html#asyncio.loop.set_task_factory "asyncio.loop.set_task_factory").
  * [gh-128388](https://github.com/python/cpython/issues/128388): Fix `PyREPL` on Windows to support more keybindings, like the `Control`-`←` and `Control`-`→` word-skipping keybindings and those with meta (i.e. `Alt`), e.g. `Alt`-`d` to `kill-word` or `Alt`-`Backspace` `backward-kill-word`.
  * [gh-126037](https://github.com/python/cpython/issues/126037): [`xml.etree.ElementTree`](https://docs.python.org/3/library/xml.etree.elementtree.html#module-xml.etree.ElementTree "xml.etree.ElementTree: Implementation of the ElementTree API."): Fix a crash in [`Element.find`](https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.find "xml.etree.ElementTree.Element.find"), [`Element.findtext`](https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.findtext "xml.etree.ElementTree.Element.findtext") and [`Element.findall`](https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.findall "xml.etree.ElementTree.Element.findall") when the tag to find implements an [`__eq__()`](https://docs.python.org/3/reference/datamodel.html#object.__eq__ "object.__eq__") method mutating the element being queried. Patch by Bénédikt Tran.
  * [gh-127712](https://github.com/python/cpython/issues/127712): Fix handling of the `secure` argument of [`logging.handlers.SMTPHandler`](https://docs.python.org/3/library/logging.handlers.html#logging.handlers.SMTPHandler "logging.handlers.SMTPHandler").
  * [gh-126033](https://github.com/python/cpython/issues/126033): [`xml.etree.ElementTree`](https://docs.python.org/3/library/xml.etree.elementtree.html#module-xml.etree.ElementTree "xml.etree.ElementTree: Implementation of the ElementTree API."): Fix a crash in [`Element.remove`](https://docs.python.org/3/library/xml.etree.elementtree.html#xml.etree.ElementTree.Element.remove "xml.etree.ElementTree.Element.remove") when the element is concurrently mutated. Patch by Bénédikt Tran.
  * [gh-118201](https://github.com/python/cpython/issues/118201): Fixed intermittent failures of [`os.confstr`](https://docs.python.org/3/library/os.html#os.confstr "os.confstr"), [`os.pathconf`](https://docs.python.org/3/library/os.html#os.pathconf "os.pathconf") and [`os.sysconf`](https://docs.python.org/3/library/os.html#os.sysconf "os.sysconf") on iOS and Android.
  * [gh-124927](https://github.com/python/cpython/issues/124927): Non-printing characters are now properly handled in the new REPL.
