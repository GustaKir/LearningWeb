Title: Isolating Extensions in Python
URL: https://docs.python.org/3/howto/isolating-extensions.html
Summary: This document discusses methods for isolating extensions in Python, including handling module state, variations between heap and static types, and the garbage-collection protocol. It includes subsections on defining heap types, changing static to heap types, and specifics on the `tp_traverse` and `tp_dealloc` functions.
