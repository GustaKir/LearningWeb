# Download Python 3.13 Documentation
**Last updated on: Apr 27, 2025 (05:38 UTC).**
To download an archive containing all the documents for this version of Python in one of various formats, follow one of links in this table.
Format | Packed as .zip | Packed as .tar.bz2  
---|---|---  
PDF | [Download](https://docs.python.org/3/archives/python-3.13-docs-pdf-a4.zip) (ca. 17 MiB) | [Download](https://docs.python.org/3/archives/python-3.13-docs-pdf-a4.tar.bz2) (ca. 17 MiB)  
HTML | [Download](https://docs.python.org/3/archives/python-3.13-docs-html.zip) (ca. 13 MiB) | [Download](https://docs.python.org/3/archives/python-3.13-docs-html.tar.bz2) (ca. 8 MiB)  
Plain text | [Download](https://docs.python.org/3/archives/python-3.13-docs-text.zip) (ca. 4 MiB) | [Download](https://docs.python.org/3/archives/python-3.13-docs-text.tar.bz2) (ca. 3 MiB)  
Texinfo | [Download](https://docs.python.org/3/archives/python-3.13-docs-texinfo.zip) (ca. 9 MiB) | [Download](https://docs.python.org/3/archives/python-3.13-docs-texinfo.tar.bz2) (ca. 7 MiB)  
EPUB | [Download](https://docs.python.org/3/archives/python-3.13-docs.epub) (ca. 6 MiB) |   
These archives contain all the content in the documentation.
## Unpacking
Unix users should download the .tar.bz2 archives; these are bzipped tar archives and can be handled in the usual way using tar and the bzip2 program. The [Info-ZIP](https://infozip.sourceforge.net) unzip program can be used to handle the ZIP archives if desired. The .tar.bz2 archives provide the best compression and fastest download times.
Windows users can use the ZIP archives since those are customary on that platform. These are created on Unix using the Info-ZIP zip program.
