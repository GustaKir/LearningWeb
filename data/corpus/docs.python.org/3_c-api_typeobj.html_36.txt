Title: PyNumberMethods Structure in Python C API
URL: https://docs.python.org/3/c-api/typeobj.html
Summary: The PyNumberMethods structure defines a series of function pointers for various numerical operations in Python. This includes binary functions for addition, subtraction, multiplication, and more, as well as unary functions for negation and conversion. Functions are required to check the types of their operands.
---

```
typedefstruct{

```

Note
Binary and ternary functions must check the type of all their operands, and implement the necessary conversions (at least one of the operands is an instance of the defined type). If the operation is not defined for the given operands, binary and ternary functions must return `Py_NotImplemented`, if another error occurred they must return `NULL` and set an exception.
Note
The [`nb_reserved`](https://docs.python.org/3/c-api/typeobj.html#c.PyNumberMethods.nb_reserved "PyNumberMethods.nb_reserved") field should always be `NULL`. It was previously called `nb_long`, and was renamed in Python 3.0.1.
