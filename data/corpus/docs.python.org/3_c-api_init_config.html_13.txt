Title: Python Configuration: Module Search Paths
URL: https://docs.python.org/3/c-api/init_config.html
Summary: This section details the `pythonpath_env` used for defining module search paths in Python. The paths are specified as a string in the `sys.path` format and are influenced by the `PYTHONPATH` environment variable. It also mentions the `PyWideStringList` type and provides links to relevant definitions.
---

wchar_t*pythonpath_env[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.pythonpath_env "Link to this definition")
    
Module search paths ([`sys.path`](https://docs.python.org/3/library/sys.html#sys.path "sys.path")) as a string separated by `DELIM` ([`os.pathsep`](https://docs.python.org/3/library/os.html#os.pathsep "os.pathsep")).
Set by the [`PYTHONPATH`](https://docs.python.org/3/using/cmdline.html#envvar-PYTHONPATH) environment variable.
Default: `NULL`.
Part of the [Python Path Configuration](https://docs.python.org/3/c-api/init_config.html#init-path-config) input. 

[PyWideStringList](https://docs.python.org/3/c-api/init_config.html#c.PyWideStringList "PyWideStringList")module_search_paths[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.module_search_paths "Link to this definition")


intmodule_search_paths_set[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.module_search_paths_set "Link to this definition")
    
Module search paths: [`sys.path`](https://docs.python.org/3/library/sys.html#sys.path "sys.path").
If [`module_search_paths_set`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.module_search_paths_set "PyConfig.module_search_paths_set") is equal to `0`, [`Py_InitializeFromConfig()`](https://docs.python.org/3/c-api/init.html#c.Py_InitializeFromConfig "Py_InitializeFromConfig") will replace [`module_search_paths`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.module_search_paths "PyConfig.module_search_paths") and sets [`module_search_paths_set`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.module_search_paths_set "PyConfig.module_search_paths_set") to `1`.
Default: empty list (`module_search_paths`) and `0` (`module_search_paths_set`).
Part of the [Python Path Configuration](https://docs.python.org/3/c-api/init_config.html#init-path-config) output. 

intoptimization_level[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.optimization_level "Link to this definition")
    
Compilation optimization level:
  * `0`: Peephole optimizer, set `__debug__` to `True`.
  * `1`: Level 0, remove assertions, set `__debug__` to `False`.
  * `2`: Level 1, strip docstrings.


Incremented by the [`-O`](https://docs.python.org/3/using/cmdline.html#cmdoption-O) command line option. Set to the [`PYTHONOPTIMIZE`](https://docs.python.org/3/using/cmdline.html#envvar-PYTHONOPTIMIZE) environment variable value.
Default: `0`. 

[PyWideStringList](https://docs.python.org/3/c-api/init_config.html#c.PyWideStringList "PyWideStringList")orig_argv[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.orig_argv "Link to this definition")
    
The list of the original command line arguments passed to the Python executable: [`sys.orig_argv`](https://docs.python.org/3/library/sys.html#sys.orig_argv "sys.orig_argv").
If [`orig_argv`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.orig_argv "PyConfig.orig_argv") list is empty and [`argv`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.argv "PyConfig.argv") is not a list only containing an empty string, [`PyConfig_Read()`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig_Read "PyConfig_Read") copies [`argv`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.argv "PyConfig.argv") into [`orig_argv`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.orig_argv "PyConfig.orig_argv") before modifying [`argv`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.argv "PyConfig.argv") (if [`parse_argv`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.parse_argv "PyConfig.parse_argv") is non-zero).
See also the [`argv`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.argv "PyConfig.argv") member and the [`Py_GetArgcArgv()`](https://docs.python.org/3/c-api/init_config.html#c.Py_GetArgcArgv "Py_GetArgcArgv") function.
Default: empty list.
Added in version 3.10.
