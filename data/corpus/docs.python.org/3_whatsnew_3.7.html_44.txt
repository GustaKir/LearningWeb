Title: What's New In Python 3.7
URL: https://docs.python.org/3/whatsnew/3.7.html
Summary: This document outlines the new features and improvements introduced in Python 3.7, including changes such as the default limit of 4300 digits in string form and various PEPs related to annotation evaluation, locale coercion, UTF-8 runtime mode, built-in breakpoint functionality, and enhancements to the C API for thread-local storage.
---

The default limit is 4300 digits in string form.
