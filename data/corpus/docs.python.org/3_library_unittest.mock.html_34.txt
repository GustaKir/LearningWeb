Title: unittest.mock — Mock Object Library
URL: https://docs.python.org/3/library/unittest.mock.html
Summary: This documentation covers the `unittest.mock` library in Python, which is used to create mock objects for testing. It includes a quick guide, details about the Mock class, and patching methods, facilitating the testing of functions and classes by replacing parts of your system behavior.
