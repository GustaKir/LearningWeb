# `uuid` — UUID objects according to [**RFC 4122**](https://datatracker.ietf.org/doc/html/rfc4122.html)[¶](https://docs.python.org/3/library/uuid.html#module-uuid "Link to this heading")
**Source code:** [Lib/uuid.py](https://github.com/python/cpython/tree/3.13/Lib/uuid.py)
This module provides immutable [`UUID`](https://docs.python.org/3/library/uuid.html#uuid.UUID "uuid.UUID") objects (the [`UUID`](https://docs.python.org/3/library/uuid.html#uuid.UUID "uuid.UUID") class) and the functions [`uuid1()`](https://docs.python.org/3/library/uuid.html#uuid.uuid1 "uuid.uuid1"), [`uuid3()`](https://docs.python.org/3/library/uuid.html#uuid.uuid3 "uuid.uuid3"), [`uuid4()`](https://docs.python.org/3/library/uuid.html#uuid.uuid4 "uuid.uuid4"), [`uuid5()`](https://docs.python.org/3/library/uuid.html#uuid.uuid5 "uuid.uuid5") for generating version 1, 3, 4, and 5 UUIDs as specified in [**RFC 4122**](https://datatracker.ietf.org/doc/html/rfc4122.html).
If all you want is a unique ID, you should probably call [`uuid1()`](https://docs.python.org/3/library/uuid.html#uuid.uuid1 "uuid.uuid1") or [`uuid4()`](https://docs.python.org/3/library/uuid.html#uuid.uuid4 "uuid.uuid4"). Note that [`uuid1()`](https://docs.python.org/3/library/uuid.html#uuid.uuid1 "uuid.uuid1") may compromise privacy since it creates a UUID containing the computer’s network address. [`uuid4()`](https://docs.python.org/3/library/uuid.html#uuid.uuid4 "uuid.uuid4") creates a random UUID.
Depending on support from the underlying platform, [`uuid1()`](https://docs.python.org/3/library/uuid.html#uuid.uuid1 "uuid.uuid1") may or may not return a “safe” UUID. A safe UUID is one which is generated using synchronization methods that ensure no two processes can obtain the same UUID. All instances of [`UUID`](https://docs.python.org/3/library/uuid.html#uuid.UUID "uuid.UUID") have an [`is_safe`](https://docs.python.org/3/library/uuid.html#uuid.UUID.is_safe "uuid.UUID.is_safe") attribute which relays any information about the UUID’s safety, using this enumeration: 

_class_ uuid.SafeUUID[¶](https://docs.python.org/3/library/uuid.html#uuid.SafeUUID "Link to this definition")
    
Added in version 3.7. 

safe[¶](https://docs.python.org/3/library/uuid.html#uuid.SafeUUID.safe "Link to this definition")
    
The UUID was generated by the platform in a multiprocessing-safe way. 

unsafe[¶](https://docs.python.org/3/library/uuid.html#uuid.SafeUUID.unsafe "Link to this definition")
    
The UUID was not generated in a multiprocessing-safe way.
