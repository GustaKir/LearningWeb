Title: Python 3.x Changelog Enhancements
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: Updates to window attributes handling; the `wm_attributes` method now supports keyword arguments, returns a dictionary on request, and consistently returns tuples. Additionally, warnings for boolean file descriptor arguments have been added, a file modification check has been introduced in the debugger `pdb`, and updates include showing the Tcl/Tk patchlevel in `tkinter._test()`.
---

It now accepts the attribute name without the minus prefix to get window attributes and allows to specify attributes and values to set as keyword arguments. Add new optional keyword argument _return_python_dict_ : calling `w.wm_attributes(return_python_dict=True)` returns the attributes as a dict instead of a tuple. Calling `w.wm_attributes()` now returns a tuple instead of string if _wantobjects_ was set to 0.
  * [gh-82626](https://github.com/python/cpython/issues/82626): Many functions now emit a warning if a boolean value is passed as a file descriptor argument.
  * [gh-111051](https://github.com/python/cpython/issues/111051): Added check for file modification during debugging with [`pdb`](https://docs.python.org/3/library/pdb.html#module-pdb "pdb: The Python debugger for interactive interpreters.")
  * [gh-110345](https://github.com/python/cpython/issues/110345): Show the Tcl/Tk patchlevel (rather than version) in `tkinter._test()`.
  * [gh-38807](https://github.com/python/cpython/issues/38807): Fix race condition in [`trace`](https://docs.python.org/3/library/trace.html#module-trace "trace: Trace or track Python statement execution."). Instead of checking if a directory exists and creating it, directly call [`os.makedirs()`](https://docs.python.org/3/library/os.html#os.makedirs "os.makedirs") with the kwarg `exist_ok=True`.
  * [gh-75705](https://github.com/python/cpython/issues/75705): Set unixfrom envelope in [`mailbox.mbox`](https://docs.python.org/3/library/mailbox.html#mailbox.mbox "mailbox.mbox") and [`mailbox.MMDF`](https://docs.python.org/3/library/mailbox.html#mailbox.MMDF "mailbox.MMDF").
  * [gh-106233](https://github.com/python/cpython/issues/106233): Fix stacklevel in `InvalidTZPathWarning` during [`zoneinfo`](https://docs.python.org/3/library/zoneinfo.html#module-zoneinfo "zoneinfo: IANA time zone support") module import.
  * [gh-105102](https://github.com/python/cpython/issues/105102): Allow [`ctypes.Union`](https://docs.python.org/3/library/ctypes.html#ctypes.Union "ctypes.Union") to be nested in [`ctypes.Structure`](https://docs.python.org/3/library/ctypes.html#ctypes.Structure "ctypes.Structure") when the system endianness is the opposite of the classes.
  * [gh-104282](https://github.com/python/cpython/issues/104282): Fix null pointer dereference in `lzma._decode_filter_properties()` due to improper handling of BCJ filters with properties of zero length. Patch by Radislav Chugunov.
  * [gh-96471](https://github.com/python/cpython/issues/96471): Add [`queue.Queue`](https://docs.python.org/3/library/queue.html#queue.Queue "queue.Queue") termination with [`shutdown()`](https://docs.python.org/3/library/queue.html#queue.Queue.shutdown "queue.Queue.shutdown").
  * [gh-101599](https://github.com/python/cpython/issues/101599): Changed argparse flag options formatting to remove redundancy.
  * [gh-85984](https://github.com/python/cpython/issues/85984): Add POSIX pseudo-terminal functions [`os.posix_openpt()`](https://docs.python.org/3/library/os.html#os.posix_openpt "os.posix_openpt"), [`os.grantpt()`](https://docs.python.org/3/library/os.html#os.grantpt "os.grantpt"), [`os.unlockpt()`](https://docs.python.org/3/library/os.html#os.unlockpt "os.unlockpt"), and [`os.ptsname()`](https://docs.python.org/3/library/os.html#os.ptsname "os.ptsname").
  * [gh-102512](https://github.com/python/cpython/issues/102512): When [`os.fork()`](https://docs.python.org/3/library/os.html#os.fork "os.fork") is called from a foreign thread (aka `_DummyThread`), the type of the thread in a child process is changed to `_MainThread`. Also changed its name and daemonic status, it can be now joined.
  * [gh-88569](https://github.com/python/cpython/issues/88569): Add [`os.path.isreserved()`](https://docs.python.org/3/library/os.path.html#os.path.isreserved "os.path.isreserved"), which identifies reserved pathnames such as “NUL”, “AUX” and “CON”. This function is only available on Windows.
Deprecate [`pathlib.PurePath.is_reserved()`](https://docs.python.org/3/library/pathlib.html#pathlib.PurePath.is_reserved "pathlib.PurePath.is_reserved").
  * [bpo-38364](https://bugs.python.org/issue?@action=redirect&bpo=38364): The `inspect` functions `isgeneratorfunction`, `iscoroutinefunction`, `isasyncgenfunction` now support `functools.partialmethod` wrapped functions the same way they support `functools.partial`.
