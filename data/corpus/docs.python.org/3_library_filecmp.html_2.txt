Title: Filecmp Module: Directory Comparison Attributes
URL: https://docs.python.org/3/library/filecmp.html
Summary: This section explains the 'funny_files' attribute which identifies files present in both directories but not comparable, and the 'subdirs' attribute, which maps directory names in 'common_dirs' to their corresponding 'dircmp' instances, highlighting changes made in version 3.10 regarding subclass compatibility.
---

funny_files[¶](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp.funny_files "Link to this definition")
    
Files which are in both _a_ and _b_ , but could not be compared. 

subdirs[¶](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp.subdirs "Link to this definition")
    
A dictionary mapping names in [`common_dirs`](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp.common_dirs "filecmp.dircmp.common_dirs") to [`dircmp`](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp "filecmp.dircmp") instances (or MyDirCmp instances if this instance is of type MyDirCmp, a subclass of [`dircmp`](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp "filecmp.dircmp")).
Changed in version 3.10: Previously entries were always [`dircmp`](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp "filecmp.dircmp") instances. Now entries are the same type as _self_ , if _self_ is a subclass of [`dircmp`](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp "filecmp.dircmp"). 

filecmp.DEFAULT_IGNORES[¶](https://docs.python.org/3/library/filecmp.html#filecmp.DEFAULT_IGNORES "Link to this definition")
    
Added in version 3.4.
List of directories ignored by [`dircmp`](https://docs.python.org/3/library/filecmp.html#filecmp.dircmp "filecmp.dircmp") by default.
Here is a simplified example of using the `subdirs` attribute to search recursively through two directories to show common different files:
>>>```
>>> fromfilecmpimport dircmp
>>> defprint_diff_files(dcmp):
...   for name in dcmp.diff_files:
...     print("diff_file %s found in %s and %s" % (name, dcmp.left,
...        dcmp.right))
...   for sub_dcmp in dcmp.subdirs.values():
...     print_diff_files(sub_dcmp)
...
>>> dcmp = dircmp('dir1', 'dir2')
>>> print_diff_files(dcmp)

```
