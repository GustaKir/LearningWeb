# Tuple Objects[¶](https://docs.python.org/3/c-api/tuple.html#tuple-objects "Link to this heading") 

typePyTupleObject[¶](https://docs.python.org/3/c-api/tuple.html#c.PyTupleObject "Link to this definition")
    
This subtype of [`PyObject`](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject") represents a Python tuple object. 

[PyTypeObject](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject")PyTuple_Type[¶](https://docs.python.org/3/c-api/tuple.html#c.PyTuple_Type "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
This instance of [`PyTypeObject`](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject") represents the Python tuple type; it is the same object as [`tuple`](https://docs.python.org/3/library/stdtypes.html#tuple "tuple") in the Python layer. 

intPyTuple_Check([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*p)[¶](https://docs.python.org/3/c-api/tuple.html#c.PyTuple_Check "Link to this definition")
    
Return true if _p_ is a tuple object or an instance of a subtype of the tuple type. This function always succeeds. 

intPyTuple_CheckExact([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*p)[¶](https://docs.python.org/3/c-api/tuple.html#c.PyTuple_CheckExact "Link to this definition")
    
Return true if _p_ is a tuple object, but not an instance of a subtype of the tuple type. This function always succeeds. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyTuple_New([Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")len)[¶](https://docs.python.org/3/c-api/tuple.html#c.PyTuple_New "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Return a new tuple object of size _len_ , or `NULL` with an exception set on failure. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyTuple_Pack([Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")n, ...)[¶](https://docs.python.org/3/c-api/tuple.html#c.PyTuple_Pack "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Return a new tuple object of size _n_ , or `NULL` with an exception set on failure. The tuple values are initialized to the subsequent _n_ C arguments pointing to Python objects. `PyTuple_Pack(2, a, b)` is equivalent to `Py_BuildValue("(OO)", a, b)`.
