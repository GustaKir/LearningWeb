Title: Python 3 Changelog Updates
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: The changelog highlights several updates, including the modified behavior of the statistics.geometric_mean() function to return zero for datasets with a zero, the addition of new constants to the syslog module on macOS, a fix in asyncio's SubprocessTransport.close() to handle setuid executables without raising PermissionError, and new constants added to the termios module.
---

Patch by Diego Russo.
  * [gh-112540](https://github.com/python/cpython/issues/112540): The statistics.geometric_mean() function now returns zero for datasets containing a zero. Formerly, it would raise an exception.
  * [gh-87286](https://github.com/python/cpython/issues/87286): Added `LOG_FTP`, `LOG_NETINFO`, `LOG_REMOTEAUTH`, `LOG_INSTALL`, `LOG_RAS`, and `LOG_LAUNCHD` tot the [`syslog`](https://docs.python.org/3/library/syslog.html#module-syslog "syslog: An interface to the Unix syslog library routines. \(Unix\)") module, all of them constants on used on macOS.
  * [gh-112800](https://github.com/python/cpython/issues/112800): Fix [`asyncio`](https://docs.python.org/3/library/asyncio.html#module-asyncio "asyncio: Asynchronous I/O.") `SubprocessTransport.close()` not to throw `PermissionError` when used with setuid executables.
  * [gh-51944](https://github.com/python/cpython/issues/51944): Add the following constants to the [`termios`](https://docs.python.org/3/library/termios.html#module-termios "termios: POSIX style tty control. \(Unix\)") module. These values are present in macOS system headers: `ALTWERASE`, `B14400`, `B28800`, `B7200`, `B76800`, `CCAR_OFLOW`, `CCTS_OFLOW`, `CDSR_OFLOW`, `CDTR_IFLOW`, `CIGNORE`, `CRTS_IFLOW`, `EXTPROC`, `IUTF8`, `MDMBUF`, `NL2`, `NL3`, `NOKERNINFO`, `ONOEOT`, `OXTABS`, `VDSUSP`, `VSTATUS`.
  * [gh-79325](https://github.com/python/cpython/issues/79325): Fix an infinite recursion error in [`tempfile.TemporaryDirectory()`](https://docs.python.org/3/library/tempfile.html#tempfile.TemporaryDirectory "tempfile.TemporaryDirectory") cleanup on Windows.
  * [gh-94692](https://github.com/python/cpython/issues/94692): [`shutil.rmtree()`](https://docs.python.org/3/library/shutil.html#shutil.rmtree "shutil.rmtree") now only catches OSError exceptions. Previously a symlink attack resistant version of `shutil.rmtree()` could ignore or pass to the error handler arbitrary exception when invalid arguments were provided.
  * [gh-112736](https://github.com/python/cpython/issues/112736): The use of del-safe symbols in `subprocess` was refactored to allow for use in cross-platform build environments.
  * [gh-112727](https://github.com/python/cpython/issues/112727): Speed up [`pathlib.Path.absolute()`](https://docs.python.org/3/library/pathlib.html#pathlib.Path.absolute "pathlib.Path.absolute"). Patch by Barney Gale.
  * [gh-74690](https://github.com/python/cpython/issues/74690): Speedup [`issubclass()`](https://docs.python.org/3/library/functions.html#issubclass "issubclass") checks against simple [`runtime-checkable protocols`](https://docs.python.org/3/library/typing.html#typing.runtime_checkable "typing.runtime_checkable") by around 6%. Patch by Alex Waygood.
  * [gh-74690](https://github.com/python/cpython/issues/74690): Speedup [`isinstance()`](https://docs.python.org/3/library/functions.html#isinstance "isinstance") checks by roughly 20% for [`runtime-checkable protocols`](https://docs.python.org/3/library/typing.html#typing.runtime_checkable "typing.runtime_checkable") that only have one callable member. Speedup [`issubclass()`](https://docs.python.org/3/library/functions.html#issubclass "issubclass") checks for these protocols by roughly 10%. Patch by Alex Waygood.
  * [gh-112645](https://github.com/python/cpython/issues/112645): Remove deprecation error on passing `onerror` to [`shutil.rmtree()`](https://docs.python.org/3/library/shutil.html#shutil.rmtree "shutil.rmtree").
  * [gh-112640](https://github.com/python/cpython/issues/112640): Add `kwdefaults` parameter to [`types.FunctionType`](https://docs.python.org/3/library/types.html#types.FunctionType "types.FunctionType") to set default keyword argument values.
  * [gh-112622](https://github.com/python/cpython/issues/112622): Ensure `name` parameter is passed to event loop in [`asyncio.create_task()`](https://docs.python.org/3/library/asyncio-task.html#asyncio.create_task "asyncio.create_task").
  * [gh-112618](https://github.com/python/cpython/issues/112618): Fix a caching bug relating to [`typing.Annotated`](https://docs.python.org/3/library/typing.html#typing.Annotated "typing.Annotated"). `Annotated[str, True]` is no longer identical to `Annotated[str, 1]`.
  * [gh-112334](https://github.com/python/cpython/issues/112334): Fixed a performance regression in 3.12â€™s [`subprocess`](https://docs.python.org/3/library/subprocess.html#module-subprocess "subprocess: Subprocess management.") on Linux where it would no longer use the fast-path `vfork()` system call when it could have due to a logic bug, instead falling back to the safe but slower `fork()`.
Also fixed a second 3.12.0 potential security bug.
