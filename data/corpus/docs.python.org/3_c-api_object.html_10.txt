Title: PyType_GetTypeDataSize Function
URL: https://docs.python.org/3/c-api/object.html
Summary: The PyType_GetTypeDataSize function returns the size of the memory reserved for an instance of the given class (cls). This size can exceed the value specified in PyType_Spec.basicsize and is safe to use with functions like memset. This function is part of the Stable ABI since version 3.12.
---

[Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")PyType_GetTypeDataSize([PyTypeObject](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject")*cls)[¶](https://docs.python.org/3/c-api/object.html#c.PyType_GetTypeDataSize "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable) since version 3.12._
Return the size of the instance memory space reserved for _cls_ , i.e. the size of the memory [`PyObject_GetTypeData()`](https://docs.python.org/3/c-api/object.html#c.PyObject_GetTypeData "PyObject_GetTypeData") returns.
This may be larger than requested using [`-PyType_Spec.basicsize`](https://docs.python.org/3/c-api/type.html#c.PyType_Spec.basicsize "PyType_Spec.basicsize"); it is safe to use this larger size (e.g. with `memset()`).
The type _cls_ **must** have been created using negative [`PyType_Spec.basicsize`](https://docs.python.org/3/c-api/type.html#c.PyType_Spec.basicsize "PyType_Spec.basicsize"). Python does not check this.
On error, set an exception and return a negative value.
Added in version 3.12.

Get a pointer to per-item data for a class with [`Py_TPFLAGS_ITEMS_AT_END`](https://docs.python.org/3/c-api/typeobj.html#c.Py_TPFLAGS_ITEMS_AT_END "Py_TPFLAGS_ITEMS_AT_END").
On error, set an exception and return `NULL`. [`TypeError`](https://docs.python.org/3/library/exceptions.html#TypeError "TypeError") is raised if _o_ does not have [`Py_TPFLAGS_ITEMS_AT_END`](https://docs.python.org/3/c-api/typeobj.html#c.Py_TPFLAGS_ITEMS_AT_END "Py_TPFLAGS_ITEMS_AT_END") set.
Added in version 3.12.
