Title: Pathlib - Object-oriented Filesystem Paths
URL: https://docs.python.org/3/library/pathlib.html
Summary: The `pathlib` module in Python provides an object-oriented interface to interact with filesystem paths. This section includes an overview of basic usage, exceptions, pure paths, their properties, operators, and methods for accessing individual components of paths.
