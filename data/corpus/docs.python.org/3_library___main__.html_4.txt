Title: __main__ Scope in Python
URL: https://docs.python.org/3/library/__main__.html
Summary: This section discusses the `__main__` scope in Python, noting that it lacks a `__file__` attribute in interactive mode. It mentions its usage in implementing the Python debugger (`pdb`) and identifier completion (`rlcompleter`), along with providing a table of contents linking to related topics.

Note that in this case the `__main__` scope doesn’t contain a `__file__` attribute as it’s interactive.
The `__main__` scope is used in the implementation of [`pdb`](https://docs.python.org/3/library/pdb.html#module-pdb "pdb: The Python debugger for interactive interpreters.") and [`rlcompleter`](https://docs.python.org/3/library/rlcompleter.html#module-rlcompleter "rlcompleter: Python identifier completion, suitable for the GNU readline library.").
