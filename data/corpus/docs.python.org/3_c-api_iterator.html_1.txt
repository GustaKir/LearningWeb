Title: PyCallIter_New Function
URL: https://docs.python.org/3/c-api/iterator.html
Summary: The PyCallIter_New function creates a new iterator in Python. It takes a callable object as the first parameter, which should return the next value in the iteration. The iteration ends when the callable returns a value equal to the provided sentinel.
---

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyCallIter_New([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*callable, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*sentinel)[¶](https://docs.python.org/3/c-api/iterator.html#c.PyCallIter_New "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Return a new iterator. The first parameter, _callable_ , can be any Python callable object that can be called with no parameters; each call to it should return the next item in the iteration. When _callable_ returns a value equal to _sentinel_ , the iteration will be terminated.
