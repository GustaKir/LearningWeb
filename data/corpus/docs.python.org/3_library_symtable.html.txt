# `symtable` — Access to the compiler’s symbol tables[¶](https://docs.python.org/3/library/symtable.html#module-symtable "Link to this heading")
**Source code:** [Lib/symtable.py](https://github.com/python/cpython/tree/3.13/Lib/symtable.py)
Symbol tables are generated by the compiler from AST just before bytecode is generated. The symbol table is responsible for calculating the scope of every identifier in the code. [`symtable`](https://docs.python.org/3/library/symtable.html#module-symtable "symtable: Interface to the compiler's internal symbol tables.") provides an interface to examine these tables.
## Generating Symbol Tables[¶](https://docs.python.org/3/library/symtable.html#generating-symbol-tables "Link to this heading") 

symtable.symtable(_code_ , _filename_ , _compile_type_)[¶](https://docs.python.org/3/library/symtable.html#symtable.symtable "Link to this definition")
    
Return the toplevel [`SymbolTable`](https://docs.python.org/3/library/symtable.html#symtable.SymbolTable "symtable.SymbolTable") for the Python source _code_. _filename_ is the name of the file containing the code. _compile_type_ is like the _mode_ argument to [`compile()`](https://docs.python.org/3/library/functions.html#compile "compile").
## Examining Symbol Tables[¶](https://docs.python.org/3/library/symtable.html#examining-symbol-tables "Link to this heading") 

_class_ symtable.SymbolTableType[¶](https://docs.python.org/3/library/symtable.html#symtable.SymbolTableType "Link to this definition")
    
An enumeration indicating the type of a [`SymbolTable`](https://docs.python.org/3/library/symtable.html#symtable.SymbolTable "symtable.SymbolTable") object. 

MODULE _= "module"_[¶](https://docs.python.org/3/library/symtable.html#symtable.SymbolTableType.MODULE "Link to this definition")
    
Used for the symbol table of a module. 

FUNCTION _= "function"_[¶](https://docs.python.org/3/library/symtable.html#symtable.SymbolTableType.FUNCTION "Link to this definition")
    
Used for the symbol table of a function. 

CLASS _= "class"_[¶](https://docs.python.org/3/library/symtable.html#symtable.SymbolTableType.CLASS "Link to this definition")
    
Used for the symbol table of a class.
The following members refer to different flavors of [annotation scopes](https://docs.python.org/3/reference/executionmodel.html#annotation-scopes). 

ANNOTATION _= "annotation"_[¶](https://docs.python.org/3/library/symtable.html#symtable.SymbolTableType.ANNOTATION "Link to this definition")
    
Used for annotations if `from __future__ import annotations` is active.
