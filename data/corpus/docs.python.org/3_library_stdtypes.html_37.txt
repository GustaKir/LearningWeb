Title: Byte and Bytearray Methods in Python
URL: https://docs.python.org/3/library/stdtypes.html
Summary: This section discusses methods for the `bytes` and `bytearray` types in Python, specifically `isascii()`, `isdigit()`, and `islower()`. The `isascii()` method checks if all bytes are ASCII, while `isdigit()` assesses if all bytes are ASCII decimal digits. It clarifies the conditions under which these methods return `True` or `False`.
---

```


bytes.isascii()[¶](https://docs.python.org/3/library/stdtypes.html#bytes.isascii "Link to this definition")


bytearray.isascii()[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.isascii "Link to this definition")
    
Return `True` if the sequence is empty or all bytes in the sequence are ASCII, `False` otherwise. ASCII bytes are in the range 0-0x7F.
Added in version 3.7. 

bytes.isdigit()[¶](https://docs.python.org/3/library/stdtypes.html#bytes.isdigit "Link to this definition")


bytearray.isdigit()[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.isdigit "Link to this definition")
    
Return `True` if all bytes in the sequence are ASCII decimal digits and the sequence is not empty, `False` otherwise. ASCII decimal digits are those byte values in the sequence `b'0123456789'`.
For example:
>>>```
>>> b'1234'.isdigit()
True
>>> b'1.23'.isdigit()
False

```


bytes.islower()[¶](https://docs.python.org/3/library/stdtypes.html#bytes.islower "Link to this definition")


bytearray.islower()[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.islower "Link to this definition")
    
Return `True` if there is at least one lowercase ASCII character in the sequence and no uppercase ASCII characters, `False` otherwise.
For example:
>>>```
>>> b'hello world'.islower()
True
>>> b'Hello world'.islower()
False

```

Lowercase ASCII characters are those byte values in the sequence `b'abcdefghijklmnopqrstuvwxyz'`. Uppercase ASCII characters are those byte values in the sequence `b'ABCDEFGHIJKLMNOPQRSTUVWXYZ'`. 

bytes.isspace()[¶](https://docs.python.org/3/library/stdtypes.html#bytes.isspace "Link to this definition")


bytearray.isspace()[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.isspace "Link to this definition")
    
Return `True` if all bytes in the sequence are ASCII whitespace and the sequence is not empty, `False` otherwise. ASCII whitespace characters are those byte values in the sequence `b' \t\n\r\x0b\f'` (space, tab, newline, carriage return, vertical tab, form feed). 

bytes.istitle()[¶](https://docs.python.org/3/library/stdtypes.html#bytes.istitle "Link to this definition")


bytearray.istitle()[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.istitle "Link to this definition")
    
Return `True` if the sequence is ASCII titlecase and the sequence is not empty, `False` otherwise. See [`bytes.title()`](https://docs.python.org/3/library/stdtypes.html#bytes.title "bytes.title") for more details on the definition of “titlecase”.
For example:
>>>```
>>> b'Hello World'.istitle()
True
>>> b'Hello world'.istitle()
False

```


bytes.isupper()[¶](https://docs.python.org/3/library/stdtypes.html#bytes.isupper "Link to this definition")


bytearray.isupper()[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.isupper "Link to this definition")
    
Return `True` if there is at least one uppercase alphabetic ASCII character in the sequence and no lowercase ASCII characters, `False` otherwise.
For example:
>>>```
>>> b'HELLO WORLD'.isupper()
True
>>> b'Hello world'.isupper()
False

```

Lowercase ASCII characters are those byte values in the sequence `b'abcdefghijklmnopqrstuvwxyz'`. Uppercase ASCII characters are those byte values in the sequence `b'ABCDEFGHIJKLMNOPQRSTUVWXYZ'`. 

bytes.lower()[¶](https://docs.python.org/3/library/stdtypes.html#bytes.lower "Link to this definition")


bytearray.lower()[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.lower "Link to this definition")
    
Return a copy of the sequence with all the uppercase ASCII characters converted to their corresponding lowercase counterpart.
For example:
>>>```
>>> b'Hello World'.lower()
b'hello world'

```

Lowercase ASCII characters are those byte values in the sequence `b'abcdefghijklmnopqrstuvwxyz'`. Uppercase ASCII characters are those byte values in the sequence `b'ABCDEFGHIJKLMNOPQRSTUVWXYZ'`.
Note
The bytearray version of this method does _not_ operate in place - it always produces a new object, even if no changes were made. 

bytes.splitlines(_keepends =False_)[¶](https://docs.python.org/3/library/stdtypes.html#bytes.splitlines "Link to this definition")


bytearray.splitlines(_keepends =False_)[¶](https://docs.python.org/3/library/stdtypes.html#bytearray.splitlines "Link to this definition")
    
Return a list of the lines in the binary sequence, breaking at ASCII line boundaries. This method uses the [universal newlines](https://docs.python.org/3/glossary.html#term-universal-newlines) approach to splitting lines. Line breaks are not included in the resulting list unless _keepends_ is given and true.
For example:
>>>```
>>> b'ab c\n\nde fg\rkl\r\n'.splitlines()
[b'ab c', b'', b'de fg', b'kl']
>>> b'ab c\n\nde fg\rkl\r\n'.splitlines(keepends=True)
[b'ab c\n', b'\n', b'de fg\r', b'kl\r\n']
