# Object Protocol[¶](https://docs.python.org/3/c-api/object.html#object-protocol "Link to this heading") 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*Py_GetConstant(unsignedintconstant_id)[¶](https://docs.python.org/3/c-api/object.html#c.Py_GetConstant "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable) since version 3.13._
Get a [strong reference](https://docs.python.org/3/glossary.html#term-strong-reference) to a constant.
Set an exception and return `NULL` if _constant_id_ is invalid.
_constant_id_ must be one of these constant identifiers:
Constant Identifier | Value | Returned object  
---|---|--- 

Py_CONSTANT_NONE[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_NONE "Link to this definition")
| `0` | [`None`](https://docs.python.org/3/library/constants.html#None "None") 

Py_CONSTANT_FALSE[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_FALSE "Link to this definition")
| `1` | [`False`](https://docs.python.org/3/library/constants.html#False "False") 

Py_CONSTANT_TRUE[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_TRUE "Link to this definition")
| `2` | [`True`](https://docs.python.org/3/library/constants.html#True "True") 

Py_CONSTANT_ELLIPSIS[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_ELLIPSIS "Link to this definition")
| `3` | [`Ellipsis`](https://docs.python.org/3/library/constants.html#Ellipsis "Ellipsis") 

Py_CONSTANT_NOT_IMPLEMENTED[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_NOT_IMPLEMENTED "Link to this definition")
| `4` | [`NotImplemented`](https://docs.python.org/3/library/constants.html#NotImplemented "NotImplemented") 

Py_CONSTANT_ZERO[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_ZERO "Link to this definition")
| `5` | `0` 

Py_CONSTANT_ONE[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_ONE "Link to this definition")
| `6` | `1` 

Py_CONSTANT_EMPTY_STR[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_EMPTY_STR "Link to this definition")
| `7` | `''` 

Py_CONSTANT_EMPTY_BYTES[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_EMPTY_BYTES "Link to this definition")
| `8` | `b''` 

Py_CONSTANT_EMPTY_TUPLE[¶](https://docs.python.org/3/c-api/object.html#c.Py_CONSTANT_EMPTY_TUPLE "Link to this definition")
| `9` | `()`  
Numeric values are only given for projects which cannot use the constant identifiers.
Added in version 3.13.
**CPython implementation detail:** In CPython, all of these constants are [immortal](https://docs.python.org/3/glossary.html#term-immortal).
