Title: Configuration Options for Building Python
URL: https://docs.python.org/3/using/configure.html
Summary: This section outlines various configuration options for building Python, including paths for the `pkg-config` utility and commands related to the C compiler, such as the compiler command, compiler flags, and preprocessor command.
---

PKG_CONFIG[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-PKG_CONFIG "Link to this definition")
    
Path to `pkg-config` utility. 

PKG_CONFIG_LIBDIR[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-PKG_CONFIG_LIBDIR "Link to this definition")


PKG_CONFIG_PATH[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-PKG_CONFIG_PATH "Link to this definition")
    
`pkg-config` options.
### 3.3.2. C compiler options[¶](https://docs.python.org/3/using/configure.html#c-compiler-options "Link to this heading") 

CC[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-CC "Link to this definition")
    
C compiler command. 

CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-CFLAGS "Link to this definition")
    
C compiler flags. 

CPP[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-CPP "Link to this definition")
    
C preprocessor command. 

CPPFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-CPPFLAGS "Link to this definition")
    
C preprocessor flags, e.g. `-I_include_dir_`.
### 3.3.3. Linker options[¶](https://docs.python.org/3/using/configure.html#linker-options "Link to this heading") 

LDFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LDFLAGS "Link to this definition")
    
Linker flags, e.g. `-L_library_directory_`. 

LIBS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBS "Link to this definition")
    
Libraries to pass to the linker, e.g. `-l_library_`. 

MACHDEP[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-MACHDEP "Link to this definition")
    
Name for machine-dependent library files.
### 3.3.4. Options for third-party dependencies[¶](https://docs.python.org/3/using/configure.html#options-for-third-party-dependencies "Link to this heading")
Added in version 3.11. 

BZIP2_CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-BZIP2_CFLAGS "Link to this definition")


BZIP2_LIBS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-BZIP2_LIBS "Link to this definition")
    
C compiler and linker flags to link Python to `libbz2`, used by [`bz2`](https://docs.python.org/3/library/bz2.html#module-bz2 "bz2: Interfaces for bzip2 compression and decompression.") module, overriding `pkg-config`. 

CURSES_CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-CURSES_CFLAGS "Link to this definition")


CURSES_LIBS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-CURSES_LIBS "Link to this definition")
    
C compiler and linker flags for `libncurses` or `libncursesw`, used by [`curses`](https://docs.python.org/3/library/curses.html#module-curses "curses: An interface to the curses library, providing portable terminal handling. \(Unix\)") module, overriding `pkg-config`. 

GDBM_CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-GDBM_CFLAGS "Link to this definition")


GDBM_LIBS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-GDBM_LIBS "Link to this definition")
    
C compiler and linker flags for `gdbm`. 

LIBB2_CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBB2_CFLAGS "Link to this definition")


LIBB2_LIBS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBB2_LIBS "Link to this definition")
    
C compiler and linker flags for `libb2` ([BLAKE2](https://docs.python.org/3/library/hashlib.html#hashlib-blake2)), used by [`hashlib`](https://docs.python.org/3/library/hashlib.html#module-hashlib "hashlib: Secure hash and message digest algorithms.") module, overriding `pkg-config`. 

LIBEDIT_CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBEDIT_CFLAGS "Link to this definition")


LIBEDIT_LIBS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBEDIT_LIBS "Link to this definition")
    
C compiler and linker flags for `libedit`, used by [`readline`](https://docs.python.org/3/library/readline.html#module-readline "readline: GNU readline support for Python. \(Unix\)") module, overriding `pkg-config`. 

LIBFFI_CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBFFI_CFLAGS "Link to this definition")


LIBFFI_LIBS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBFFI_LIBS "Link to this definition")
    
C compiler and linker flags for `libffi`, used by [`ctypes`](https://docs.python.org/3/library/ctypes.html#module-ctypes "ctypes: A foreign function library for Python.") module, overriding `pkg-config`. 

LIBMPDEC_CFLAGS[¶](https://docs.python.org/3/using/configure.html#cmdoption-arg-LIBMPDEC_CFLAGS "Link to this definition")
