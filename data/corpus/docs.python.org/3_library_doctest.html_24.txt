Title: Footnotes and Navigation in Doctest Documentation
URL: https://docs.python.org/3/library/doctest.html
Summary: This section highlights that examples containing both expected output and exceptions are unsupported due to potential confusion. It also provides a navigation table of contents for the `doctest` documentation.
---

```

Footnotes
[[1](https://docs.python.org/3/library/doctest.html#id1)]
Examples containing both expected output and an exception are not supported. Trying to guess where one ends and the other begins is too error-prone, and that also makes for a confusing test.
