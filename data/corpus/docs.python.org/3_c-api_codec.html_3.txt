Title: PyCodec_NameReplaceErrors Function
URL: https://docs.python.org/3/c-api/codec.html
Summary: The PyCodec_NameReplaceErrors function replaces unicode encode errors with `\N{...}` escapes. It is part of the Stable ABI since version 3.7 and was added in version 3.5. This function returns a new reference.
---

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyCodec_NameReplaceErrors([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*exc)[¶](https://docs.python.org/3/c-api/codec.html#c.PyCodec_NameReplaceErrors "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable) since version 3.7._
Replace the unicode encode error with `\N{...}` escapes.
Added in version 3.5.
