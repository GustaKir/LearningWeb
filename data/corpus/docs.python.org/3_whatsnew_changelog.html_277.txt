Title: Python 3 Changelog Highlights
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: This section details several changes in Python 3, including logging improvements for warnings, updates to `typing.get_type_hints()` for forward references, and the addition of a `DeprecationWarning` for `LegacyInterpolation`, indicating its future removal in Python 3.13.
---

`*tuple[int, str]`.
  * [bpo-46557](https://bugs.python.org/issue?@action=redirect&bpo=46557): Warnings captured by the logging module are now logged without a format string to prevent systems that group logs by the msg argument from grouping captured warnings together.
  * [bpo-41370](https://bugs.python.org/issue?@action=redirect&bpo=41370): [`typing.get_type_hints()`](https://docs.python.org/3/library/typing.html#typing.get_type_hints "typing.get_type_hints") now supports evaluating strings as forward references in [PEP 585 generic aliases](https://docs.python.org/3/library/stdtypes.html#types-genericalias).
  * [bpo-46607](https://bugs.python.org/issue?@action=redirect&bpo=46607): Add [`DeprecationWarning`](https://docs.python.org/3/library/exceptions.html#DeprecationWarning "DeprecationWarning") to `LegacyInterpolation`, deprecated in the docstring since Python 3.2. Will be removed in Python 3.13. Use `BasicInterpolation` or `ExtendedInterpolation` instead.
  * [bpo-26120](https://bugs.python.org/issue?@action=redirect&bpo=26120): [`pydoc`](https://docs.python.org/3/library/pydoc.html#module-pydoc "pydoc: Documentation generator and online help system.") now excludes __future__ imports from the moduleâ€™s data items.
  * [bpo-46480](https://bugs.python.org/issue?@action=redirect&bpo=46480): Add [`typing.assert_type()`](https://docs.python.org/3/library/typing.html#typing.assert_type "typing.assert_type"). Patch by Jelle Zijlstra.
  * [bpo-46421](https://bugs.python.org/issue?@action=redirect&bpo=46421): Fix a unittest issue where if the command was invoked as `python -m unittest` and the filename(s) began with a dot (.), a `ValueError` is returned.
  * [bpo-46245](https://bugs.python.org/issue?@action=redirect&bpo=46245): Add optional parameter _dir_fd_ in [`shutil.rmtree()`](https://docs.python.org/3/library/shutil.html#shutil.rmtree "shutil.rmtree").
  * [bpo-22859](https://bugs.python.org/issue?@action=redirect&bpo=22859): `unittest.TestProgram.usageExit()` is marked as deprecated, to be removed in Python 3.13.
  * [bpo-46170](https://bugs.python.org/issue?@action=redirect&bpo=46170): Improve the error message when you try to subclass an instance of [`typing.NewType`](https://docs.python.org/3/library/typing.html#typing.NewType "typing.NewType").
  * [bpo-40296](https://bugs.python.org/issue?@action=redirect&bpo=40296): Fix supporting generic aliases in [`pydoc`](https://docs.python.org/3/library/pydoc.html#module-pydoc "pydoc: Documentation generator and online help system.").
  * [bpo-20392](https://bugs.python.org/issue?@action=redirect&bpo=20392): Fix inconsistency with uppercase file extensions in `MimeTypes.guess_type()`.
