Title: New Modules in Python 3.3
URL: https://docs.python.org/3/whatsnew/3.3.html
Summary: Python 3.3 introduces two new modules: `ipaddress`, which enables creation and manipulation of IPv4 and IPv6 addresses and networks, and `lzma`, which offers data compression and decompression capabilities using the LZMA algorithm and supports `.xz` and `.lzma` file formats.

### ipaddress[¶](https://docs.python.org/3/whatsnew/3.3.html#ipaddress "Link to this heading")
The new [`ipaddress`](https://docs.python.org/3/library/ipaddress.html#module-ipaddress "ipaddress: IPv4/IPv6 manipulation library.") module provides tools for creating and manipulating objects representing IPv4 and IPv6 addresses, networks and interfaces (i.e. an IP address associated with a specific IP subnet).
(Contributed by Google and Peter Moody in [**PEP 3144**](https://peps.python.org/pep-3144/).)
### lzma[¶](https://docs.python.org/3/whatsnew/3.3.html#lzma "Link to this heading")
The newly added [`lzma`](https://docs.python.org/3/library/lzma.html#module-lzma "lzma: A Python wrapper for the liblzma compression library.") module provides data compression and decompression using the LZMA algorithm, including support for the `.xz` and `.lzma` file formats.
(Contributed by Nadeem Vawda and Per Øyvind Karlsen in [bpo-6715](https://bugs.python.org/issue?@action=redirect&bpo=6715).)
## Improved Modules[¶](https://docs.python.org/3/whatsnew/3.3.html#improved-modules "Link to this heading")
### abc[¶](https://docs.python.org/3/whatsnew/3.3.html#abc "Link to this heading")
Improved support for abstract base classes containing descriptors composed with abstract methods. The recommended approach to declaring abstract descriptors is now to provide `__isabstractmethod__` as a dynamically updated property. The built-in descriptors have been updated accordingly.
  * [`abc.abstractproperty`](https://docs.python.org/3/library/abc.html#abc.abstractproperty "abc.abstractproperty") has been deprecated, use [`property`](https://docs.python.org/3/library/functions.html#property "property") with [`abc.abstractmethod()`](https://docs.python.org/3/library/abc.html#abc.abstractmethod "abc.abstractmethod") instead.
  * [`abc.abstractclassmethod`](https://docs.python.org/3/library/abc.html#abc.abstractclassmethod "abc.abstractclassmethod") has been deprecated, use [`classmethod`](https://docs.python.org/3/library/functions.html#classmethod "classmethod") with [`abc.abstractmethod()`](https://docs.python.org/3/library/abc.html#abc.abstractmethod "abc.abstractmethod") instead.
  * [`abc.abstractstaticmethod`](https://docs.python.org/3/library/abc.html#abc.abstractstaticmethod "abc.abstractstaticmethod") has been deprecated, use [`staticmethod`](https://docs.python.org/3/library/functions.html#staticmethod "staticmethod") with [`abc.abstractmethod()`](https://docs.python.org/3/library/abc.html#abc.abstractmethod "abc.abstractmethod") instead.
