Title: Locale Configuration in Python C API
URL: https://docs.python.org/3/c-api/init_config.html
Summary: This section outlines the configuration options for locale handling in the Python C API. It details the 'configure_locale' setting to adjust the LC_CTYPE locale based on user preference, with the default value of '1' for Python configs and '0' for isolated ones. It also describes the 'coerce_c_locale' setting, which dictates whether to coerce the C locale based on the LC_CTYPE locale value.
---

intconfigure_locale[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.configure_locale "Link to this definition")
    
Set the LC_CTYPE locale to the user preferred locale.
If equals to `0`, set [`coerce_c_locale`](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.coerce_c_locale "PyPreConfig.coerce_c_locale") and [`coerce_c_locale_warn`](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.coerce_c_locale_warn "PyPreConfig.coerce_c_locale_warn") members to `0`.
See the [locale encoding](https://docs.python.org/3/glossary.html#term-locale-encoding).
Default: `1` in Python config, `0` in isolated config. 

intcoerce_c_locale[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.coerce_c_locale "Link to this definition")
    
If equals to `2`, coerce the C locale.
If equals to `1`, read the LC_CTYPE locale to decide if it should be coerced.
See the [locale encoding](https://docs.python.org/3/glossary.html#term-locale-encoding).
Default: `-1` in Python config, `0` in isolated config. 

intcoerce_c_locale_warn[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.coerce_c_locale_warn "Link to this definition")
    
If non-zero, emit a warning if the C locale is coerced.
Default: `-1` in Python config, `0` in isolated config. 

intdev_mode[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.dev_mode "Link to this definition")
    
[Python Development Mode](https://docs.python.org/3/library/devmode.html#devmode): see [`PyConfig.dev_mode`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.dev_mode "PyConfig.dev_mode").
Default: `-1` in Python mode, `0` in isolated mode. 

intisolated[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.isolated "Link to this definition")
    
Isolated mode: see [`PyConfig.isolated`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.isolated "PyConfig.isolated").
Default: `0` in Python mode, `1` in isolated mode. 

intlegacy_windows_fs_encoding[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.legacy_windows_fs_encoding "Link to this definition")
    
If non-zero:
  * Set [`PyPreConfig.utf8_mode`](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.utf8_mode "PyPreConfig.utf8_mode") to `0`,
  * Set [`PyConfig.filesystem_encoding`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.filesystem_encoding "PyConfig.filesystem_encoding") to `"mbcs"`,
  * Set [`PyConfig.filesystem_errors`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.filesystem_errors "PyConfig.filesystem_errors") to `"replace"`.


Initialized from the [`PYTHONLEGACYWINDOWSFSENCODING`](https://docs.python.org/3/using/cmdline.html#envvar-PYTHONLEGACYWINDOWSFSENCODING) environment variable value.
Only available on Windows. `#ifdef MS_WINDOWS` macro can be used for Windows specific code.
Default: `0`. 

intparse_argv[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.parse_argv "Link to this definition")
    
If non-zero, [`Py_PreInitializeFromArgs()`](https://docs.python.org/3/c-api/init_config.html#c.Py_PreInitializeFromArgs "Py_PreInitializeFromArgs") and [`Py_PreInitializeFromBytesArgs()`](https://docs.python.org/3/c-api/init_config.html#c.Py_PreInitializeFromBytesArgs "Py_PreInitializeFromBytesArgs") parse their `argv` argument the same way the regular Python parses command line arguments: see [Command Line Arguments](https://docs.python.org/3/using/cmdline.html#using-on-cmdline).
Default: `1` in Python config, `0` in isolated config. 

intuse_environment[¶](https://docs.python.org/3/c-api/init_config.html#c.PyPreConfig.use_environment "Link to this definition")
    
Use [environment variables](https://docs.python.org/3/using/cmdline.html#using-on-envvars)? See [`PyConfig.use_environment`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.use_environment "PyConfig.use_environment").
Default: `1` in Python config and `0` in isolated config.
