Title: C API: Object Formatting and Representation
URL: https://docs.python.org/3/c-api/object.html
Summary: This section describes the functions `PyObject_Format` and `PyObject_Repr` in the Python C API. `PyObject_Format` formats a given PyObject using a specified format specification, returning a formatted string or NULL on failure. `PyObject_Repr` provides a string representation of the PyObject.
---

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyObject_Format([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*obj, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*format_spec)[¶](https://docs.python.org/3/c-api/object.html#c.PyObject_Format "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Format _obj_ using _format_spec_. This is equivalent to the Python expression `format(obj, format_spec)`.
_format_spec_ may be `NULL`. In this case the call is equivalent to `format(obj)`. Returns the formatted string on success, `NULL` on failure. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyObject_Repr([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*o)[¶](https://docs.python.org/3/c-api/object.html#c.PyObject_Repr "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Compute a string representation of object _o_. Returns the string representation on success, `NULL` on failure. This is the equivalent of the Python expression `repr(o)`. Called by the [`repr()`](https://docs.python.org/3/library/functions.html#repr "repr") built-in function.
Changed in version 3.4: This function now includes a debug assertion to help ensure that it does not silently discard an active exception. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyObject_ASCII([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*o)[¶](https://docs.python.org/3/c-api/object.html#c.PyObject_ASCII "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
As [`PyObject_Repr()`](https://docs.python.org/3/c-api/object.html#c.PyObject_Repr "PyObject_Repr"), compute a string representation of object _o_ , but escape the non-ASCII characters in the string returned by [`PyObject_Repr()`](https://docs.python.org/3/c-api/object.html#c.PyObject_Repr "PyObject_Repr") with `\x`, `\u` or `\U` escapes. This generates a string similar to that returned by [`PyObject_Repr()`](https://docs.python.org/3/c-api/object.html#c.PyObject_Repr "PyObject_Repr") in Python 2. Called by the [`ascii()`](https://docs.python.org/3/library/functions.html#ascii "ascii") built-in function. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyObject_Str([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*o)[¶](https://docs.python.org/3/c-api/object.html#c.PyObject_Str "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Compute a string representation of object _o_. Returns the string representation on success, `NULL` on failure. This is the equivalent of the Python expression `str(o)`. Called by the [`str()`](https://docs.python.org/3/library/stdtypes.html#str "str") built-in function and, therefore, by the [`print()`](https://docs.python.org/3/library/functions.html#print "print") function.
Changed in version 3.4: This function now includes a debug assertion to help ensure that it does not silently discard an active exception. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyObject_Bytes([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*o)[¶](https://docs.python.org/3/c-api/object.html#c.PyObject_Bytes "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Compute a bytes representation of object _o_. `NULL` is returned on failure and a bytes object on success. This is equivalent to the Python expression `bytes(o)`, when _o_ is not an integer. Unlike `bytes(o)`, a TypeError is raised when _o_ is an integer instead of a zero-initialized bytes object.
