# Frame Objects[¶](https://docs.python.org/3/c-api/frame.html#frame-objects "Link to this heading") 

typePyFrameObject[¶](https://docs.python.org/3/c-api/frame.html#c.PyFrameObject "Link to this definition")
    _Part of the[ Limited API](https://docs.python.org/3/c-api/stable.html#stable) (as an opaque struct)._
The C structure of the objects used to describe frame objects.
There are no public members in this structure.
Changed in version 3.11: The members of this structure were removed from the public C API. Refer to the [What’s New entry](https://docs.python.org/3/whatsnew/3.11.html#pyframeobject-3-11-hiding) for details.
The [`PyEval_GetFrame()`](https://docs.python.org/3/c-api/reflection.html#c.PyEval_GetFrame "PyEval_GetFrame") and [`PyThreadState_GetFrame()`](https://docs.python.org/3/c-api/init.html#c.PyThreadState_GetFrame "PyThreadState_GetFrame") functions can be used to get a frame object.
See also [Reflection](https://docs.python.org/3/c-api/reflection.html#reflection). 

[PyTypeObject](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject")PyFrame_Type[¶](https://docs.python.org/3/c-api/frame.html#c.PyFrame_Type "Link to this definition")
    
The type of frame objects. It is the same object as [`types.FrameType`](https://docs.python.org/3/library/types.html#types.FrameType "types.FrameType") in the Python layer.
Changed in version 3.11: Previously, this type was only available after including `<frameobject.h>`. 

intPyFrame_Check([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*obj)[¶](https://docs.python.org/3/c-api/frame.html#c.PyFrame_Check "Link to this definition")
    
Return non-zero if _obj_ is a frame object.
Changed in version 3.11: Previously, this function was only available after including `<frameobject.h>`. 

[PyFrameObject](https://docs.python.org/3/c-api/frame.html#c.PyFrameObject "PyFrameObject")*PyFrame_GetBack([PyFrameObject](https://docs.python.org/3/c-api/frame.html#c.PyFrameObject "PyFrameObject")*frame)[¶](https://docs.python.org/3/c-api/frame.html#c.PyFrame_GetBack "Link to this definition")
    _Return value: New reference._
Get the _frame_ next outer frame.
Return a [strong reference](https://docs.python.org/3/glossary.html#term-strong-reference), or `NULL` if _frame_ has no outer frame.
Added in version 3.9.
