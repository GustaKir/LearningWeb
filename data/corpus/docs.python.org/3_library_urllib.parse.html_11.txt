Title: URL Parsing in Python's urllib.parse
URL: https://docs.python.org/3/library/urllib.parse.html
Summary: This section covers the parsing requirements for mailto URL schemes as defined in RFC 2368, rules for combining absolute and relative URLs from RFC 1808, and the formal syntax and semantics of absolute URLs as specified in RFC 1738. It also includes a table of contents for the urllib.parse module.
---

[**RFC 2368**](https://datatracker.ietf.org/doc/html/rfc2368.html) - The mailto URL scheme.
    
Parsing requirements for mailto URL schemes. 

[**RFC 1808**](https://datatracker.ietf.org/doc/html/rfc1808.html) - Relative Uniform Resource Locators
    
This Request For Comments includes the rules for joining an absolute and a relative URL, including a fair number of “Abnormal Examples” which govern the treatment of border cases. 

[**RFC 1738**](https://datatracker.ietf.org/doc/html/rfc1738.html) - Uniform Resource Locators (URL)
    
This specifies the formal syntax and semantics of absolute URLs.
