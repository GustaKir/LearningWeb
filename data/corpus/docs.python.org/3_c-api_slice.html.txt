# Slice Objects[¶](https://docs.python.org/3/c-api/slice.html#slice-objects "Link to this heading") 

[PyTypeObject](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject")PySlice_Type[¶](https://docs.python.org/3/c-api/slice.html#c.PySlice_Type "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
The type object for slice objects. This is the same as [`slice`](https://docs.python.org/3/library/functions.html#slice "slice") in the Python layer. 

intPySlice_Check([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*ob)[¶](https://docs.python.org/3/c-api/slice.html#c.PySlice_Check "Link to this definition")
    
Return true if _ob_ is a slice object; _ob_ must not be `NULL`. This function always succeeds. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PySlice_New([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*start, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*stop, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*step)[¶](https://docs.python.org/3/c-api/slice.html#c.PySlice_New "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Return a new slice object with the given values. The _start_ , _stop_ , and _step_ parameters are used as the values of the slice object attributes of the same names. Any of the values may be `NULL`, in which case the `None` will be used for the corresponding attribute.
Return `NULL` with an exception set if the new object could not be allocated. 

intPySlice_GetIndices([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*slice, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")length, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")*start, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")*stop, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")*step)[¶](https://docs.python.org/3/c-api/slice.html#c.PySlice_GetIndices "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Retrieve the start, stop and step indices from the slice object _slice_ , assuming a sequence of length _length_. Treats indices greater than _length_ as errors.
Returns `0` on success and `-1` on error with no exception set (unless one of the indices was not `None` and failed to be converted to an integer, in which case `-1` is returned with an exception set).
You probably do not want to use this function.
Changed in version 3.2: The parameter type for the _slice_ parameter was `PySliceObject*` before.
