Title: Tkinter Ttk: Child Elements Specification
URL: https://docs.python.org/3/library/tkinter.ttk.html
Summary: This section describes the '_children_' attribute, which specifies a list of elements to be placed inside a Tkinter Ttk element. Each child is described as a tuple where the first item indicates the layout name and subsequent items represent the layout specifications.
---

_children_ : [sublayout… ]
    
Specifies a list of elements to place inside the element. Each element is a tuple (or other sequence type) where the first item is the layout name, and the other is a [Layout](https://docs.python.org/3/library/tkinter.ttk.html#layouts).
