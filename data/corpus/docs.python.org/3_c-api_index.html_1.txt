Title: Python/C API Reference Manual
URL: https://docs.python.org/3/c-api/index.html
Summary: This manual provides documentation for the API used by C and C++ programmers to write extension modules or embed Python. It serves as a companion to the 'Extending and Embedding the Python Interpreter' documentation, offering detailed information about API functions along with a variety of topics including coding standards, include files, useful macros, and specifics about objects, types, reference counts, and exceptions.
