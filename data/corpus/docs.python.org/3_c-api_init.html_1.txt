Title: Python Initialization and Configuration
URL: https://docs.python.org/3/c-api/init.html
Summary: This section discusses the initialization of the Python interpreter within applications using the Python/C API. It emphasizes that the `Py_Initialize()` function must be called before any other API functions, along with details about functions that can be safely invoked prior to initialization.
---

# Initialization, Finalization, and Threads[¶](https://docs.python.org/3/c-api/init.html#initialization-finalization-and-threads "Link to this heading")
See [Python Initialization Configuration](https://docs.python.org/3/c-api/init_config.html#init-config) for details on how to configure the interpreter prior to initialization.
## Before Python Initialization[¶](https://docs.python.org/3/c-api/init.html#before-python-initialization "Link to this heading")
In an application embedding Python, the [`Py_Initialize()`](https://docs.python.org/3/c-api/init.html#c.Py_Initialize "Py_Initialize") function must be called before using any other Python/C API functions; with the exception of a few functions and the [global configuration variables](https://docs.python.org/3/c-api/init.html#global-conf-vars).
The following functions can be safely called before Python is initialized:
  * Functions that initialize the interpreter:
    * [`Py_Initialize()`](https://docs.python.org/3/c-api/init.html#c.Py_Initialize "Py_Initialize")
    * [`Py_InitializeEx()`](https://docs.python.org/3/c-api/init.html#c.Py_InitializeEx "Py_InitializeEx")
    * [`Py_InitializeFromConfig()`](https://docs.python.org/3/c-api/init.html#c.Py_InitializeFromConfig "Py_InitializeFromConfig")
    * [`Py_BytesMain()`](https://docs.python.org/3/c-api/init.html#c.Py_BytesMain "Py_BytesMain")
    * [`Py_Main()`](https://docs.python.org/3/c-api/init.html#c.Py_Main "Py_Main")
    * the runtime pre-initialization functions covered in [Python Initialization Configuration](https://docs.python.org/3/c-api/init_config.html#init-config)
  * Configuration functions:
    * [`PyImport_AppendInittab()`](https://docs.python.org/3/c-api/import.html#c.PyImport_AppendInittab "PyImport_AppendInittab")
    * [`PyImport_ExtendInittab()`](https://docs.python.org/3/c-api/import.html#c.PyImport_ExtendInittab "PyImport_ExtendInittab")
    * `PyInitFrozenExtensions()`
    * [`PyMem_SetAllocator()`](https://docs.python.org/3/c-api/memory.html#c.PyMem_SetAllocator "PyMem_SetAllocator")
    * [`PyMem_SetupDebugHooks()`](https://docs.python.org/3/c-api/memory.html#c.PyMem_SetupDebugHooks "PyMem_SetupDebugHooks")
    * [`PyObject_SetArenaAllocator()`](https://docs.python.org/3/c-api/memory.html#c.PyObject_SetArenaAllocator "PyObject_SetArenaAllocator")
    * [`Py_SetProgramName()`](https://docs.python.org/3/c-api/init.html#c.Py_SetProgramName "Py_SetProgramName")
    * [`Py_SetPythonHome()`](https://docs.python.org/3/c-api/init.html#c.Py_SetPythonHome "Py_SetPythonHome")
    * [`PySys_ResetWarnOptions()`](https://docs.python.org/3/c-api/sys.html#c.PySys_ResetWarnOptions "PySys_ResetWarnOptions")
    * the configuration functions covered in [Python Initialization Configuration](https://docs.python.org/3/c-api/init_config.html#init-config)
  * Informative functions:
    * [`Py_IsInitialized()`](https://docs.python.org/3/c-api/init.html#c.Py_IsInitialized "Py_IsInitialized")
    * [`PyMem_GetAllocator()`](https://docs.python.org/3/c-api/memory.html#c.PyMem_GetAllocator "PyMem_GetAllocator")
    * [`PyObject_GetArenaAllocator()`](https://docs.python.org/3/c-api/memory.html#c.PyObject_GetArenaAllocator "PyObject_GetArenaAllocator")
    * [`Py_GetBuildInfo()`](https://docs.python.org/3/c-api/init.html#c.Py_GetBuildInfo "Py_GetBuildInfo")
    * [`Py_GetCompiler()`](https://docs.python.org/3/c-api/init.html#c.Py_GetCompiler "Py_GetCompiler")
    * [`Py_GetCopyright()`](https://docs.python.org/3/c-api/init.html#c.Py_GetCopyright "Py_GetCopyright")
    * [`Py_GetPlatform()`](https://docs.python.org/3/c-api/init.html#c.Py_GetPlatform "Py_GetPlatform")
    * [`Py_GetVersion()`](https://docs.python.org/3/c-api/init.html#c.Py_GetVersion "Py_GetVersion")
    * [`Py_IsInitialized()`](https://docs.python.org/3/c-api/init.html#c.Py_IsInitialized "Py_IsInitialized")
  * Utilities:
    * [`Py_DecodeLocale()`](https://docs.python.org/3/c-api/sys.html#c.Py_DecodeLocale "Py_DecodeLocale")
    * the status reporting and utility functions covered in [Python Initialization Configuration](https://docs.python.org/3/c-api/init_config.html#init-config)
  * Memory allocators:
    * [`PyMem_RawMalloc()`](https://docs.python.org/3/c-api/memory.html#c.PyMem_RawMalloc "PyMem_RawMalloc")
    * [`PyMem_RawRealloc()`](https://docs.python.org/3/c-api/memory.html#c.PyMem_RawRealloc "PyMem_RawRealloc")
    * [`PyMem_RawCalloc()`](https://docs.python.org/3/c-api/memory.html#c.PyMem_RawCalloc "PyMem_RawCalloc")
    * [`PyMem_RawFree()`](https://docs.python.org/3/c-api/memory.html#c.PyMem_RawFree "PyMem_RawFree")
  * Synchronization:
    * [`PyMutex_Lock()`](https://docs.python.org/3/c-api/init.html#c.PyMutex_Lock "PyMutex_Lock")
    * [`PyMutex_Unlock()`](https://docs.python.org/3/c-api/init.html#c.PyMutex_Unlock "PyMutex_Unlock")
