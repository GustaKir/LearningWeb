Title: Python 3.9 Changelog Highlights
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: This section covers key updates in Python 3.9, including compatibility improvements for the Unicode dictionary with subinterpreters, adjustments to the `_PyUnicode_FromId()` function, handling of empty strings in `platform.libc_ver()`, and a change in behavior for `randrange()` regarding TypeError when a step is specified without a stop argument.

  * [bpo-38308](https://bugs.python.org/issue?@action=redirect&bpo=38308): Add optional _weights_ to _statistics.harmonic_mean()_.

It will not be set as the default root window and will be destroyed right after closing the dialog window. It will help to use these simple dialog windows in programs which do not need other GUI.
  * [bpo-25246](https://bugs.python.org/issue?@action=redirect&bpo=25246): Optimized [`collections.deque.remove()`](https://docs.python.org/3/library/collections.html#collections.deque.remove "collections.deque.remove").
  * [bpo-35728](https://bugs.python.org/issue?@action=redirect&bpo=35728): Added a root parameter to [`tkinter.font.nametofont()`](https://docs.python.org/3/library/tkinter.font.html#tkinter.font.nametofont "tkinter.font.nametofont").
  * [bpo-15303](https://bugs.python.org/issue?@action=redirect&bpo=15303): [`tkinter`](https://docs.python.org/3/library/tkinter.html#module-tkinter "tkinter: Interface to Tcl/Tk for graphical user interfaces") supports now widgets with boolean value False.
  * [bpo-42681](https://bugs.python.org/issue?@action=redirect&bpo=42681): Fixed range checks for color and pair numbers in [`curses`](https://docs.python.org/3/library/curses.html#module-curses "curses: An interface to the curses library, providing portable terminal handling. \(Unix\)").
