Title: PyUnicode_Translate Function
URL: https://docs.python.org/3/c-api/unicode.html
Summary: The `PyUnicode_Translate` function translates a string by applying a character mapping table, returning the resulting Unicode object or `NULL` if an exception occurs. The mapping table can map Unicode ordinal integers or `None` to denote character deletion, and should implement the `__getitem__()` interface.
---

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyUnicode_Translate([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*unicode, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*table, constchar*errors)[¶](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_Translate "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Translate a string by applying a character mapping table to it and return the resulting Unicode object. Return `NULL` if an exception was raised by the codec.
The mapping table must map Unicode ordinal integers to Unicode ordinal integers or `None` (causing deletion of the character).
Mapping tables need only provide the [`__getitem__()`](https://docs.python.org/3/reference/datamodel.html#object.__getitem__ "object.__getitem__") interface; dictionaries and sequences work well. Unmapped character ordinals (ones which cause a [`LookupError`](https://docs.python.org/3/library/exceptions.html#LookupError "LookupError")) are left untouched and are copied as-is.
_errors_ has the usual meaning for codecs. It may be `NULL` which indicates to use the default error handling.
### MBCS codecs for Windows[¶](https://docs.python.org/3/c-api/unicode.html#mbcs-codecs-for-windows "Link to this heading")
These are the MBCS codec APIs. They are currently only available on Windows and use the Win32 MBCS converters to implement the conversions. Note that MBCS (or DBCS) is a class of encodings, not just one. The target encoding is defined by the user settings on the machine running the codec.
