Title: doctest — Test Interactive Python Examples
URL: https://docs.python.org/3/library/doctest.html
Summary: The `doctest` module allows for testing interactive examples in Python docstrings and text files. It provides simple usage methods for checking examples and includes command-line functionality, helping to ensure code correctness by validating embedded examples.
