Title: Handling Exceptions in Python
URL: https://docs.python.org/3/tutorial/errors.html
Summary: This section demonstrates how to raise an `ExceptionGroup` with sub-exceptions in Python. It includes an example where an `OSError` is raised three times in a loop, and each occurrence is noted and collected before encapsulating them into a single `ExceptionGroup` for handling multiple exceptions more effectively.
---

```
>>> deff():
...   raise OSError('operation failed')
...
>>> excs = []
>>> for i in range(3):
...   try:
...     f()
...   except Exception as e:
...     e.add_note(f'Happened in Iteration {i+1}')
...     excs.append(e)
...
>>> raise ExceptionGroup('We have some problems', excs)
```
