Title: Introduction to sys.monitoring
URL: https://docs.python.org/3/library/sys.monitoring.html
Summary: The `sys.monitoring` module provides functionality for execution event monitoring in Python. It includes tool identifiers, event types (local, ancillary, and others), mechanisms for turning events on and off, and instructions for registering and using monitoring tools.
