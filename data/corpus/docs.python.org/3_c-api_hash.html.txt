# PyHash API[¶](https://docs.python.org/3/c-api/hash.html#pyhash-api "Link to this heading")
See also the [`PyTypeObject.tp_hash`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_hash "PyTypeObject.tp_hash") member and [Hashing of numeric types](https://docs.python.org/3/library/stdtypes.html#numeric-hash). 

typePy_hash_t[¶](https://docs.python.org/3/c-api/hash.html#c.Py_hash_t "Link to this definition")
    
Hash value type: signed integer.
Added in version 3.2. 

typePy_uhash_t[¶](https://docs.python.org/3/c-api/hash.html#c.Py_uhash_t "Link to this definition")
    
Hash value type: unsigned integer.
Added in version 3.2. 

PyHASH_MODULUS[¶](https://docs.python.org/3/c-api/hash.html#c.PyHASH_MODULUS "Link to this definition")
    
The [Mersenne prime](https://en.wikipedia.org/wiki/Mersenne_prime) `P = 2**n -1`, used for numeric hash scheme.
Added in version 3.13. 

PyHASH_BITS[¶](https://docs.python.org/3/c-api/hash.html#c.PyHASH_BITS "Link to this definition")
    
The exponent `n` of `P` in [`PyHASH_MODULUS`](https://docs.python.org/3/c-api/hash.html#c.PyHASH_MODULUS "PyHASH_MODULUS").
Added in version 3.13. 

PyHASH_MULTIPLIER[¶](https://docs.python.org/3/c-api/hash.html#c.PyHASH_MULTIPLIER "Link to this definition")
    
Prime multiplier used in string and various other hashes.
Added in version 3.13. 

PyHASH_INF[¶](https://docs.python.org/3/c-api/hash.html#c.PyHASH_INF "Link to this definition")
    
The hash value returned for a positive infinity.
Added in version 3.13. 

PyHASH_IMAG[¶](https://docs.python.org/3/c-api/hash.html#c.PyHASH_IMAG "Link to this definition")
    
The multiplier used for the imaginary part of a complex number.
Added in version 3.13. 

typePyHash_FuncDef[¶](https://docs.python.org/3/c-api/hash.html#c.PyHash_FuncDef "Link to this definition")
    
Hash function definition used by [`PyHash_GetFuncDef()`](https://docs.python.org/3/c-api/hash.html#c.PyHash_GetFuncDef "PyHash_GetFuncDef"). 

constchar*name[¶](https://docs.python.org/3/c-api/hash.html#c.PyHash_FuncDef.name "Link to this definition")
    
Hash function name (UTF-8 encoded string). 

constinthash_bits[¶](https://docs.python.org/3/c-api/hash.html#c.PyHash_FuncDef.hash_bits "Link to this definition")
    
Internal size of the hash value in bits. 

constintseed_bits[¶](https://docs.python.org/3/c-api/hash.html#c.PyHash_FuncDef.seed_bits "Link to this definition")
    
Size of seed input in bits.
Added in version 3.4. 

[PyHash_FuncDef](https://docs.python.org/3/c-api/hash.html#c.PyHash_FuncDef "PyHash_FuncDef")*PyHash_GetFuncDef(void)[¶](https://docs.python.org/3/c-api/hash.html#c.PyHash_GetFuncDef "Link to this definition")
    
Get the hash function definition.
See also
[**PEP 456**](https://peps.python.org/pep-0456/) “Secure and interchangeable hash algorithm”.
Added in version 3.4.
