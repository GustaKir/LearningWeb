Title: Overview of IDLE in Python
URL: https://docs.python.org/3/library/idle.html
Summary: This documentation chunk discusses the private nature of the idlelib code and its ability to backport feature changes. It includes a link to PEP 434 and a table of contents for the IDLE documentation, outlining various menus available in both the Shell and Editor.
---

Except for files listed under ‘Startup’, the idlelib code is ‘private’ in sense that feature changes can be backported (see [**PEP 434**](https://peps.python.org/pep-0434/)).
