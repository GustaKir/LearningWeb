# Dictionary Objects[¶](https://docs.python.org/3/c-api/dict.html#dictionary-objects "Link to this heading") 

typePyDictObject[¶](https://docs.python.org/3/c-api/dict.html#c.PyDictObject "Link to this definition")
    
This subtype of [`PyObject`](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject") represents a Python dictionary object. 

[PyTypeObject](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject")PyDict_Type[¶](https://docs.python.org/3/c-api/dict.html#c.PyDict_Type "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
This instance of [`PyTypeObject`](https://docs.python.org/3/c-api/type.html#c.PyTypeObject "PyTypeObject") represents the Python dictionary type. This is the same object as [`dict`](https://docs.python.org/3/library/stdtypes.html#dict "dict") in the Python layer. 

intPyDict_Check([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*p)[¶](https://docs.python.org/3/c-api/dict.html#c.PyDict_Check "Link to this definition")
    
Return true if _p_ is a dict object or an instance of a subtype of the dict type. This function always succeeds. 

intPyDict_CheckExact([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*p)[¶](https://docs.python.org/3/c-api/dict.html#c.PyDict_CheckExact "Link to this definition")
    
Return true if _p_ is a dict object, but not an instance of a subtype of the dict type. This function always succeeds. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyDict_New()[¶](https://docs.python.org/3/c-api/dict.html#c.PyDict_New "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Return a new empty dictionary, or `NULL` on failure. 

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyDictProxy_New([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*mapping)[¶](https://docs.python.org/3/c-api/dict.html#c.PyDictProxy_New "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Return a [`types.MappingProxyType`](https://docs.python.org/3/library/types.html#types.MappingProxyType "types.MappingProxyType") object for a mapping which enforces read-only behavior. This is normally used to create a view to prevent modification of the dictionary for non-dynamic class types. 

voidPyDict_Clear([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*p)[¶](https://docs.python.org/3/c-api/dict.html#c.PyDict_Clear "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Empty an existing dictionary of all key-value pairs.
