Title: Memory-Mapped File Flags in Python
URL: https://docs.python.org/3/library/mmap.html
Summary: This section outlines the various flags available for the `mmap.mmap()` function, including `MAP_UNIX03`, `MAP_ALIGNED_SUPER`, and `MAP_CONCEAL`, which are platform-specific. It notes the introduction of `MAP_POPULATE` in version 3.10, `MAP_STACK` in version 3.11, and further additions in version 3.12.
---

mmap.MAP_UNIX03[¶](https://docs.python.org/3/library/mmap.html#mmap.MAP_UNIX03 "Link to this definition")
    
These are the various flags that can be passed to [`mmap.mmap()`](https://docs.python.org/3/library/mmap.html#mmap.mmap "mmap.mmap"). [`MAP_ALIGNED_SUPER`](https://docs.python.org/3/library/mmap.html#mmap.MAP_ALIGNED_SUPER "mmap.MAP_ALIGNED_SUPER") is only available at FreeBSD and [`MAP_CONCEAL`](https://docs.python.org/3/library/mmap.html#mmap.MAP_CONCEAL "mmap.MAP_CONCEAL") is only available at OpenBSD. Note that some options might not be present on some systems.
Changed in version 3.10: Added [`MAP_POPULATE`](https://docs.python.org/3/library/mmap.html#mmap.MAP_POPULATE "mmap.MAP_POPULATE") constant.
Added in version 3.11: Added [`MAP_STACK`](https://docs.python.org/3/library/mmap.html#mmap.MAP_STACK "mmap.MAP_STACK") constant.
Added in version 3.12: Added [`MAP_ALIGNED_SUPER`](https://docs.python.org/3/library/mmap.html#mmap.MAP_ALIGNED_SUPER "mmap.MAP_ALIGNED_SUPER") and [`MAP_CONCEAL`](https://docs.python.org/3/library/mmap.html#mmap.MAP_CONCEAL "mmap.MAP_CONCEAL") constants.
Added in version 3.13: Added [`MAP_32BIT`](https://docs.python.org/3/library/mmap.html#mmap.MAP_32BIT "mmap.MAP_32BIT"), [`MAP_HASSEMAPHORE`](https://docs.python.org/3/library/mmap.html#mmap.MAP_HASSEMAPHORE "mmap.MAP_HASSEMAPHORE"), [`MAP_JIT`](https://docs.python.org/3/library/mmap.html#mmap.MAP_JIT "mmap.MAP_JIT"), [`MAP_NOCACHE`](https://docs.python.org/3/library/mmap.html#mmap.MAP_NOCACHE "mmap.MAP_NOCACHE"), [`MAP_NOEXTEND`](https://docs.python.org/3/library/mmap.html#mmap.MAP_NOEXTEND "mmap.MAP_NOEXTEND"), [`MAP_NORESERVE`](https://docs.python.org/3/library/mmap.html#mmap.MAP_NORESERVE "mmap.MAP_NORESERVE"), [`MAP_RESILIENT_CODESIGN`](https://docs.python.org/3/library/mmap.html#mmap.MAP_RESILIENT_CODESIGN "mmap.MAP_RESILIENT_CODESIGN"), [`MAP_RESILIENT_MEDIA`](https://docs.python.org/3/library/mmap.html#mmap.MAP_RESILIENT_MEDIA "mmap.MAP_RESILIENT_MEDIA"), [`MAP_TPRO`](https://docs.python.org/3/library/mmap.html#mmap.MAP_TPRO "mmap.MAP_TPRO"), [`MAP_TRANSLATED_ALLOW_EXECUTE`](https://docs.python.org/3/library/mmap.html#mmap.MAP_TRANSLATED_ALLOW_EXECUTE "mmap.MAP_TRANSLATED_ALLOW_EXECUTE"), and [`MAP_UNIX03`](https://docs.python.org/3/library/mmap.html#mmap.MAP_UNIX03 "mmap.MAP_UNIX03") constants.
