Title: trace — Trace or track Python statement execution
URL: https://docs.python.org/3/library/trace.html
Summary: The 'trace' module in Python provides functionality to track the execution of Python statements. It includes options for command-line usage, such as main options, modifiers, and filters, as well as a programmatic interface for integration into Python scripts.
