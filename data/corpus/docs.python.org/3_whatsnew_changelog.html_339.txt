Title: Python 3 Changelog Updates
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: This document lists recent updates and patches, including changes to `ensurepip` using `importlib.resources.files()`, the implementation of Argument Clinic in `sqlite3`, a new warning for importing deprecated `tkinter.tix`, the removal of an unused function from `inspect`, and other bug fixes.
---

Patch by RÃ©mi Lapeyre.
  * [bpo-37449](https://bugs.python.org/issue?@action=redirect&bpo=37449): `ensurepip` now uses `importlib.resources.files()` traversable APIs
  * [bpo-40956](https://bugs.python.org/issue?@action=redirect&bpo=40956): Use Argument Clinic in [`sqlite3`](https://docs.python.org/3/library/sqlite3.html#module-sqlite3 "sqlite3: A DB-API 2.0 implementation using SQLite 3.x."). Patches by Erlend E. Aasland.
  * [bpo-41730](https://bugs.python.org/issue?@action=redirect&bpo=41730): `DeprecationWarning` is now raised when importing `tkinter.tix`, which has been deprecated in documentation since Python 3.6.
  * [bpo-20684](https://bugs.python.org/issue?@action=redirect&bpo=20684): Remove unused `_signature_get_bound_param` function from [`inspect`](https://docs.python.org/3/library/inspect.html#module-inspect "inspect: Extract information and source code from live objects.") - by Anthony Sottile.
  * [bpo-41402](https://bugs.python.org/issue?@action=redirect&bpo=41402): Fix [`email.message.EmailMessage.set_content()`](https://docs.python.org/3/library/email.message.html#email.message.EmailMessage.set_content "email.message.EmailMessage.set_content") when called with binary data and `7bit` content transfer encoding.
  * [bpo-32695](https://bugs.python.org/issue?@action=redirect&bpo=32695): The _compresslevel_ and _preset_ keyword arguments of [`tarfile.open()`](https://docs.python.org/3/library/tarfile.html#tarfile.open "tarfile.open") are now both documented and tested.
  * [bpo-41137](https://bugs.python.org/issue?@action=redirect&bpo=41137): Use utf-8 encoding while reading .pdbrc files. Patch by Srinivas Reddy Thatiparthy
  * [bpo-24391](https://bugs.python.org/issue?@action=redirect&bpo=24391): Improved reprs of [`threading`](https://docs.python.org/3/library/threading.html#module-threading "threading: Thread-based parallelism.") synchronization objects: [`Semaphore`](https://docs.python.org/3/library/threading.html#threading.Semaphore "threading.Semaphore"), [`BoundedSemaphore`](https://docs.python.org/3/library/threading.html#threading.BoundedSemaphore "threading.BoundedSemaphore"), [`Event`](https://docs.python.org/3/library/threading.html#threading.Event "threading.Event") and [`Barrier`](https://docs.python.org/3/library/threading.html#threading.Barrier "threading.Barrier").
  * [bpo-5846](https://bugs.python.org/issue?@action=redirect&bpo=5846): Deprecated the following [`unittest`](https://docs.python.org/3/library/unittest.html#module-unittest "unittest: Unit testing framework for Python.") functions, scheduled for removal in Python 3.13:
    * `findTestCases()`
    * `makeSuite()`
    * `getTestCaseNames()`
Use [`TestLoader`](https://docs.python.org/3/library/unittest.html#unittest.TestLoader "unittest.TestLoader") methods instead:
    * [`unittest.TestLoader.loadTestsFromModule()`](https://docs.python.org/3/library/unittest.html#unittest.TestLoader.loadTestsFromModule "unittest.TestLoader.loadTestsFromModule")
    * [`unittest.TestLoader.loadTestsFromTestCase()`](https://docs.python.org/3/library/unittest.html#unittest.TestLoader.loadTestsFromTestCase "unittest.TestLoader.loadTestsFromTestCase")
    * [`unittest.TestLoader.getTestCaseNames()`](https://docs.python.org/3/library/unittest.html#unittest.TestLoader.getTestCaseNames "unittest.TestLoader.getTestCaseNames")
Patch by Erlend E. Aasland.
  * [bpo-40563](https://bugs.python.org/issue?@action=redirect&bpo=40563): Support pathlike objects on dbm/shelve.
