Title: Buffer API Functions in Python C API
URL: https://docs.python.org/3/c-api/buffer.html
Summary: This section describes two functions in the Python C API for handling buffer operations: `PyBuffer_FromContiguous`, which copies bytes from a contiguous memory buffer to a Python buffer view, and `PyBuffer_ToContiguous`, which copies bytes from a Python buffer view to a specified memory buffer. It covers parameters like length and memory order, and indicates success or error states.
---

intPyBuffer_FromContiguous(const[Py_buffer](https://docs.python.org/3/c-api/buffer.html#c.Py_buffer "Py_buffer")*view, constvoid*buf, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")len, charfort)[¶](https://docs.python.org/3/c-api/buffer.html#c.PyBuffer_FromContiguous "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable) since version 3.11._
Copy contiguous _len_ bytes from _buf_ to _view_. _fort_ can be `'C'` or `'F'` (for C-style or Fortran-style ordering). `0` is returned on success, `-1` on error. 

intPyBuffer_ToContiguous(void*buf, const[Py_buffer](https://docs.python.org/3/c-api/buffer.html#c.Py_buffer "Py_buffer")*src, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")len, charorder)[¶](https://docs.python.org/3/c-api/buffer.html#c.PyBuffer_ToContiguous "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable) since version 3.11._
Copy _len_ bytes from _src_ to its contiguous representation in _buf_. _order_ can be `'C'` or `'F'` or `'A'` (for C-style or Fortran-style ordering or either one). `0` is returned on success, `-1` on error.
This function fails if _len_ != _src- >len_. 

intPyObject_CopyData([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*dest, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*src)[¶](https://docs.python.org/3/c-api/buffer.html#c.PyObject_CopyData "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable) since version 3.11._
Copy data from _src_ to _dest_ buffer. Can convert between C-style and or Fortran-style buffers.
`0` is returned on success, `-1` on error. 

voidPyBuffer_FillContiguousStrides(intndims, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")*shape, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")*strides, intitemsize, charorder)[¶](https://docs.python.org/3/c-api/buffer.html#c.PyBuffer_FillContiguousStrides "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable) since version 3.11._
Fill the _strides_ array with byte-strides of a [contiguous](https://docs.python.org/3/glossary.html#term-contiguous) (C-style if _order_ is `'C'` or Fortran-style if _order_ is `'F'`) array of the given shape with the given number of bytes per element.
