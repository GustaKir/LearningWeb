Title: Deprecation of strptime() Format String Behavior
URL: https://docs.python.org/3/library/datetime.html
Summary: The `strptime()` method in Python's datetime library, which uses a format string containing a day of the month without a year, has been deprecated since version 3.13. A `DeprecationWarning` will be emitted, and future versions may change this to an error or default to a leap year.
---

```

Deprecated since version 3.13, will be removed in version 3.15: [`strptime()`](https://docs.python.org/3/library/datetime.html#datetime.datetime.strptime "datetime.datetime.strptime") calls using a format string containing a day of month without a year now emit a [`DeprecationWarning`](https://docs.python.org/3/library/exceptions.html#DeprecationWarning "DeprecationWarning"). In 3.15 or later we may change this into an error or change the default year to a leap year. See [gh-70647](https://github.com/python/cpython/issues/70647).
