Title: urllib.response.addinfourl Class Overview
URL: https://docs.python.org/3/library/urllib.request.html
Summary: The 'addinfourl' class in the 'urllib.response' module provides access to the URL of a resource retrieved, the response headers as an EmailMessage instance, and the server's status code. The 'geturl()' method can also be used to retrieve the URL.
