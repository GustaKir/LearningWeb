Title: UUID Command-Line Usage Examples
URL: https://docs.python.org/3/library/uuid.html
Summary: This section provides examples of how to use the `uuid` command line interface to generate different types of UUIDs. It includes commands for generating a random UUID (using uuid4), a UUID using uuid1, and a UUID using uuid5 with a specified name.
---

```

## Command-Line Example[¶](https://docs.python.org/3/library/uuid.html#command-line-example "Link to this heading")
Here are some examples of typical usage of the [`uuid`](https://docs.python.org/3/library/uuid.html#module-uuid "uuid: UUID objects \(universally unique identifiers\) according to RFC 4122") command line interface:
```
# generate a random uuid - by default uuid4() is used
$python-muuid
# generate a uuid using uuid1()
$python-muuid-uuuid1
# generate a uuid using uuid5
$python-muuid-uuuid5-n@url-Nexample.com

```
