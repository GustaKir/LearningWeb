Title: Python Configuration Options
URL: https://docs.python.org/3/c-api/init_config.html
Summary: This section details the configuration options for Python startup, including the 'quiet' mode that suppresses the display of copyright and version information, as well as options for running commands and filenames via command line. The 'quiet' mode is controlled by the '-q' option, while the command to execute is specified with the '-c' option.
---

intquiet[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.quiet "Link to this definition")
    
Quiet mode. If greater than `0`, don’t display the copyright and version at Python startup in interactive mode.
Incremented by the [`-q`](https://docs.python.org/3/using/cmdline.html#cmdoption-q) command line option.
Default: `0`. 

wchar_t*run_command[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.run_command "Link to this definition")
    
Value of the [`-c`](https://docs.python.org/3/using/cmdline.html#cmdoption-c) command line option.
Used by [`Py_RunMain()`](https://docs.python.org/3/c-api/init.html#c.Py_RunMain "Py_RunMain").
Default: `NULL`. 

wchar_t*run_filename[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.run_filename "Link to this definition")
    
Filename passed on the command line: trailing command line argument without [`-c`](https://docs.python.org/3/using/cmdline.html#cmdoption-c) or [`-m`](https://docs.python.org/3/using/cmdline.html#cmdoption-m). It is used by the [`Py_RunMain()`](https://docs.python.org/3/c-api/init.html#c.Py_RunMain "Py_RunMain") function.
For example, it is set to `script.py` by the `python3 script.py arg` command line.
See also the [`PyConfig.skip_source_first_line`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.skip_source_first_line "PyConfig.skip_source_first_line") option.
Default: `NULL`. 

wchar_t*run_module[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.run_module "Link to this definition")
    
Value of the [`-m`](https://docs.python.org/3/using/cmdline.html#cmdoption-m) command line option.
Used by [`Py_RunMain()`](https://docs.python.org/3/c-api/init.html#c.Py_RunMain "Py_RunMain").
Default: `NULL`. 

wchar_t*run_presite[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.run_presite "Link to this definition")
    
`package.module` path to module that should be imported before `site.py` is run.
Set by the [`-X presite=package.module`](https://docs.python.org/3/using/cmdline.html#cmdoption-X) command-line option and the [`PYTHON_PRESITE`](https://docs.python.org/3/using/cmdline.html#envvar-PYTHON_PRESITE) environment variable. The command-line option takes precedence.
Needs a [debug build of Python](https://docs.python.org/3/using/configure.html#debug-build) (the `Py_DEBUG` macro must be defined).
Default: `NULL`. 

intshow_ref_count[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.show_ref_count "Link to this definition")
    
Show total reference count at exit (excluding [immortal](https://docs.python.org/3/glossary.html#term-immortal) objects)?
Set to `1` by [`-X showrefcount`](https://docs.python.org/3/using/cmdline.html#cmdoption-X) command line option.
Needs a [debug build of Python](https://docs.python.org/3/using/configure.html#debug-build) (the `Py_REF_DEBUG` macro must be defined).
Default: `0`. 

intsite_import[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.site_import "Link to this definition")
    
Import the [`site`](https://docs.python.org/3/library/site.html#module-site "site: Module responsible for site-specific configuration.") module at startup?
If equal to zero, disable the import of the module site and the site-dependent manipulations of [`sys.path`](https://docs.python.org/3/library/sys.html#sys.path "sys.path") that it entails.
Also disable these manipulations if the [`site`](https://docs.python.org/3/library/site.html#module-site "site: Module responsible for site-specific configuration.") module is explicitly imported later (call [`site.main()`](https://docs.python.org/3/library/site.html#site.main "site.main") if you want them to be triggered).
Set to `0` by the [`-S`](https://docs.python.org/3/using/cmdline.html#cmdoption-S) command line option.
[`sys.flags.no_site`](https://docs.python.org/3/library/sys.html#sys.flags "sys.flags") is set to the inverted value of [`site_import`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.site_import "PyConfig.site_import").
Default: `1`. 

intskip_source_first_line[¶](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.skip_source_first_line "Link to this definition")
    
If non-zero, skip the first line of the [`PyConfig.run_filename`](https://docs.python.org/3/c-api/init_config.html#c.PyConfig.run_filename "PyConfig.run_filename") source.
It allows the usage of non-Unix forms of `#!cmd`. This is intended for a DOS specific hack only.
Set to `1` by the [`-x`](https://docs.python.org/3/using/cmdline.html#cmdoption-x) command line option.
Default: `0`.
