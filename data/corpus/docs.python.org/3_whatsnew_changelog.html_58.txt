Title: Changes in Python Library
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: Recent improvements in the Python library include adding a future warning for `functools.partial.__get__()`, enhancing the `__repr__()` method for `functools.partialmethod`, and fixing an issue in `argparse.ArgumentParser.parses_args()` related to the `exit_on_error` parameter.
---

### Library[¶](https://docs.python.org/3/whatsnew/changelog.html#id63 "Link to this heading")
  * [gh-121027](https://github.com/python/cpython/issues/121027): Add a future warning in `functools.partial.__get__()`. In future Python versions [`functools.partial`](https://docs.python.org/3/library/functools.html#functools.partial "functools.partial") will be a method descriptor.
  * [gh-121025](https://github.com/python/cpython/issues/121025): Improve the [`__repr__()`](https://docs.python.org/3/reference/datamodel.html#object.__repr__ "object.__repr__") of [`functools.partialmethod`](https://docs.python.org/3/library/functools.html#functools.partialmethod "functools.partialmethod"). Patch by Bénédikt Tran.
  * [gh-121018](https://github.com/python/cpython/issues/121018): Fixed an issue where `argparse.ArgumentParser.parses_args()` did not honor `exit_on_error=False` when given unrecognized arguments. Patch by Ben Hsing.
  * [gh-119614](https://github.com/python/cpython/issues/119614): Fix truncation of strings with embedded null characters in some internal operations in [`tkinter`](https://docs.python.org/3/library/tkinter.html#module-tkinter "tkinter: Interface to Tcl/Tk for graphical user interfaces").
  * [gh-120910](https://github.com/python/cpython/issues/120910): When reading installed files from an egg, use `relative_to(walk_up=True)` to honor files installed outside of the installation root.
  * [gh-120888](https://github.com/python/cpython/issues/120888): Upgrade pip wheel bundled with ensurepip (pip 24.1.1)
  * [gh-101830](https://github.com/python/cpython/issues/101830): Accessing the [`tkinter`](https://docs.python.org/3/library/tkinter.html#module-tkinter "tkinter: Interface to Tcl/Tk for graphical user interfaces") object’s string representation no longer converts the underlying Tcl object to a string on Windows.
  * [gh-120811](https://github.com/python/cpython/issues/120811): Fix possible memory leak in [`contextvars.Context.run()`](https://docs.python.org/3/library/contextvars.html#contextvars.Context.run "contextvars.Context.run").
  * [gh-120769](https://github.com/python/cpython/issues/120769): Make empty line in [`pdb`](https://docs.python.org/3/library/pdb.html#module-pdb "pdb: The Python debugger for interactive interpreters.") repeats the last command even when the command is from `cmdqueue`.
  * [gh-120732](https://github.com/python/cpython/issues/120732): Fix `name` passing to [`unittest.mock.Mock`](https://docs.python.org/3/library/unittest.mock.html#unittest.mock.Mock "unittest.mock.Mock") object when using [`unittest.mock.create_autospec()`](https://docs.python.org/3/library/unittest.mock.html#unittest.mock.create_autospec "unittest.mock.create_autospec").
  * [gh-120683](https://github.com/python/cpython/issues/120683): Fix an error in [`logging.LogRecord`](https://docs.python.org/3/library/logging.html#logging.LogRecord "logging.LogRecord"), when the integer part of the timestamp is rounded up, while the millisecond calculation truncates, causing the log timestamp to be wrong by up to 999 ms (affected roughly 1 in 8 million timestamps).
  * [gh-120633](https://github.com/python/cpython/issues/120633): Move scrollbar and remove tear-off menus in turtledemo.
  * [gh-120541](https://github.com/python/cpython/issues/120541): Improve the prompt in the “less” pager when [`help()`](https://docs.python.org/3/library/functions.html#help "help") is called with non-string argument.
  * [gh-120495](https://github.com/python/cpython/issues/120495): Fix incorrect exception handling in Tab Nanny. Patch by Wulian233.
  * [gh-120381](https://github.com/python/cpython/issues/120381): Correct [`inspect.ismethoddescriptor()`](https://docs.python.org/3/library/inspect.html#inspect.ismethoddescriptor "inspect.ismethoddescriptor") to check also for the lack of [`__delete__()`](https://docs.python.org/3/reference/datamodel.html#object.__delete__ "object.__delete__"). Patch by Jan Kaliszewski.
  * [gh-90425](https://github.com/python/cpython/issues/90425): The OS byte in gzip headers is now always set to 255 when using [`gzip.compress()`](https://docs.python.org/3/library/gzip.html#gzip.compress "gzip.compress").
  * [gh-120343](https://github.com/python/cpython/issues/120343): Fix column offset reporting for tokens that come after multiline f-strings in the [`tokenize`](https://docs.python.org/3/library/tokenize.html#module-tokenize "tokenize: Lexical scanner for Python source code.") module.
  * [gh-119600](https://github.com/python/cpython/issues/119600): Fix [`unittest.mock.patch()`](https://docs.python.org/3/library/unittest.mock.html#unittest.mock.patch "unittest.mock.patch") to not read attributes of the target when `new_callable` is set. Patch by Robert Collins.
  * [gh-114053](https://github.com/python/cpython/issues/114053): Fix edge-case bug where [`typing.get_type_hints()`](https://docs.python.org/3/library/typing.html#typing.get_type_hints "typing.get_type_hints") would produce incorrect
