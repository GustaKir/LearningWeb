Title: PyUnicode_Partition Function in Python C API
URL: https://docs.python.org/3/c-api/unicode.html
Summary: The `PyUnicode_Partition` function splits a Unicode string at the first occurrence of a specified separator, returning a 3-tuple of the parts before, including, and after the separator. If the separator isn't found, the string and two empty strings are returned. The function requires a non-empty separator and returns NULL on error.

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyUnicode_Partition([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*unicode, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*sep)[¶](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_Partition "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Split a Unicode string at the first occurrence of _sep_ , and return a 3-tuple containing the part before the separator, the separator itself, and the part after the separator. If the separator is not found, return a 3-tuple containing the string itself, followed by two empty strings.
_sep_ must not be empty.
On error, return `NULL` with an exception set.
Equivalent to [`str.partition()`](https://docs.python.org/3/library/stdtypes.html#str.partition "str.partition").

Similar to [`PyUnicode_Partition()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_Partition "PyUnicode_Partition"), but split a Unicode string at the last occurrence of _sep_. If the separator is not found, return a 3-tuple containing two empty strings, followed by the string itself.
_sep_ must not be empty.
On error, return `NULL` with an exception set.
Equivalent to [`str.rpartition()`](https://docs.python.org/3/library/stdtypes.html#str.rpartition "str.rpartition").

Return `1` if _substr_ matches `unicode[start:end]` at the given tail end (_direction_ == `-1` means to do a prefix match, _direction_ == `1` a suffix match), `0` otherwise. Return `-1` if an error occurred. 

[Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")PyUnicode_Find([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*unicode, [PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*substr, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")start, [Py_ssize_t](https://docs.python.org/3/c-api/intro.html#c.Py_ssize_t "Py_ssize_t")end, intdirection)[¶](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_Find "Link to this definition")
    _Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Return the first position of _substr_ in `unicode[start:end]` using the given _direction_ (_direction_ == `1` means to do a forward search, _direction_ == `-1` a backward search). The return value is the index of the first match; a value of `-1` indicates that no match was found, and `-2` indicates that an error occurred and an exception has been set.
