Title: configparser.MultilineContinuationError
URL: https://docs.python.org/3/library/configparser.html
Summary: This exception is raised when a key in the config parser continues with an indented line but lacks a corresponding value. It was introduced in version 3.13.
---

_exception_ configparser.MultilineContinuationError[¶](https://docs.python.org/3/library/configparser.html#configparser.MultilineContinuationError "Link to this definition")
    
Exception raised when a key without a corresponding value is continued with an indented line.
Added in version 3.13.
