Title: Python 3.3 Unicode Encoding Functions
URL: https://docs.python.org/3/whatsnew/3.3.html
Summary: This section outlines various unicode encoding functions introduced or modified in Python 3.3, including `PyUnicode_Encode()` and its alternatives like `PyUnicode_AsEncodedObject()`, as well as specific encoding methods such as `PyUnicode_EncodeUTF8()`, `PyUnicode_EncodeUTF16()`, and `PyUnicode_EncodeRawUnicodeEscape()`.
---

Encoders:
  * `PyUnicode_Encode()`: use `PyUnicode_AsEncodedObject()`
  * `PyUnicode_EncodeUTF7()`
  * `PyUnicode_EncodeUTF8()`: use [`PyUnicode_AsUTF8()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_AsUTF8 "PyUnicode_AsUTF8") or [`PyUnicode_AsUTF8String()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_AsUTF8String "PyUnicode_AsUTF8String")
  * `PyUnicode_EncodeUTF32()`
  * `PyUnicode_EncodeUTF16()`
  * `PyUnicode_EncodeUnicodeEscape()` use [`PyUnicode_AsUnicodeEscapeString()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_AsUnicodeEscapeString "PyUnicode_AsUnicodeEscapeString")
  * `PyUnicode_EncodeRawUnicodeEscape()` use [`PyUnicode_AsRawUnicodeEscapeString()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_AsRawUnicodeEscapeString "PyUnicode_AsRawUnicodeEscapeString")
  * `PyUnicode_EncodeLatin1()`: use [`PyUnicode_AsLatin1String()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_AsLatin1String "PyUnicode_AsLatin1String")
  * `PyUnicode_EncodeASCII()`: use [`PyUnicode_AsASCIIString()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_AsASCIIString "PyUnicode_AsASCIIString")
  * `PyUnicode_EncodeCharmap()`
  * `PyUnicode_TranslateCharmap()`
  * `PyUnicode_EncodeMBCS()`: use [`PyUnicode_AsMBCSString()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_AsMBCSString "PyUnicode_AsMBCSString") or [`PyUnicode_EncodeCodePage()`](https://docs.python.org/3/c-api/unicode.html#c.PyUnicode_EncodeCodePage "PyUnicode_EncodeCodePage") (with `CP_ACP` code_page)
  * `PyUnicode_EncodeDecimal()`, `PyUnicode_TransformDecimalToASCII()`
