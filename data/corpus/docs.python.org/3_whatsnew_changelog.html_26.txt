Title: Python 3.10 Changelog Updates
URL: https://docs.python.org/3/whatsnew/changelog.html
Summary: This section outlines various bug fixes and enhancements in Python 3.10, including improvements to `asyncio.TaskGroup.create_task()`, `asyncio.staggered.staggered_race()`, and `faulthandler.dump_traceback()`, as well as unification of instance checks for `typing.Union` and `types.UnionType`.
---

  * [gh-128479](https://github.com/python/cpython/issues/128479): Fix `asyncio.staggered.staggered_race()` leaking tasks and issuing an unhandled exception.
  * [gh-128400](https://github.com/python/cpython/issues/128400): Fix crash when using [`faulthandler.dump_traceback()`](https://docs.python.org/3/library/faulthandler.html#faulthandler.dump_traceback "faulthandler.dump_traceback") while other threads are active on the [free threaded](https://docs.python.org/3/glossary.html#term-free-threading) build.
  * [gh-88834](https://github.com/python/cpython/issues/88834): Unify the instance check for [`typing.Union`](https://docs.python.org/3/library/typing.html#typing.Union "typing.Union") and [`types.UnionType`](https://docs.python.org/3/library/types.html#types.UnionType "types.UnionType"): `Union` now uses the instance checks against its parameters instead of the subclass checks.
  * [gh-128302](https://github.com/python/cpython/issues/128302): Fix `xml.dom.xmlbuilder.DOMEntityResolver.resolveEntity()`, which was broken by the Python 3.0 transition.
  * [gh-128302](https://github.com/python/cpython/issues/128302): Allow `xml.dom.xmlbuilder.DOMParser.parse()` to correctly handle `xml.dom.xmlbuilder.DOMInputSource` instances that only have a `systemId` attribute set.
  * [gh-112064](https://github.com/python/cpython/issues/112064): Fix incorrect handling of negative read sizes in [`HTTPResponse.read`](https://docs.python.org/3/library/http.client.html#http.client.HTTPResponse.read "http.client.HTTPResponse.read"). Patch by Yury Manushkin.
  * [gh-58956](https://github.com/python/cpython/issues/58956): Fixed a frame reference leak in [`bdb`](https://docs.python.org/3/library/bdb.html#module-bdb "bdb: Debugger framework.").
  * [gh-128131](https://github.com/python/cpython/issues/128131): Completely support random access of uncompressed unencrypted read-only zip files obtained by [`ZipFile.open`](https://docs.python.org/3/library/zipfile.html#zipfile.ZipFile.open "zipfile.ZipFile.open").
  * [gh-112328](https://github.com/python/cpython/issues/112328): [`enum.EnumDict`](https://docs.python.org/3/library/enum.html#enum.EnumDict "enum.EnumDict") can now be used without resorting to private API.
  * [gh-127975](https://github.com/python/cpython/issues/127975): Avoid reusing quote types in [`ast.unparse()`](https://docs.python.org/3/library/ast.html#ast.unparse "ast.unparse") if not needed.
  * [gh-128062](https://github.com/python/cpython/issues/128062): Revert the font of [`turtledemo`](https://docs.python.org/3/library/turtle.html#module-turtledemo "turtledemo: A viewer for example turtle scripts")’s menu bar to its default value and display the shortcut keys in the correct position.
  * [gh-128014](https://github.com/python/cpython/issues/128014): Fix resetting the default window icon by passing `default=''` to the [`tkinter`](https://docs.python.org/3/library/tkinter.html#module-tkinter "tkinter: Interface to Tcl/Tk for graphical user interfaces") method `wm_iconbitmap()`.
  * [gh-115514](https://github.com/python/cpython/issues/115514): Fix exceptions and incomplete writes after `asyncio._SelectorTransport` is closed before writes are completed.
  * [gh-41872](https://github.com/python/cpython/issues/41872): Fix quick extraction of module docstrings from a file in [`pydoc`](https://docs.python.org/3/library/pydoc.html#module-pydoc "pydoc: Documentation generator and online help system."). It now supports docstrings with single quotes, escape sequences, raw string literals, and other Python syntax.
  * [gh-127060](https://github.com/python/cpython/issues/127060): Set TERM environment variable to “dumb” to disable traceback colors in IDLE, since IDLE doesn’t understand ANSI escape sequences. Patch by Victor Stinner.
  * [gh-126742](https://github.com/python/cpython/issues/126742): Fix support of localized error messages reported by _[dlerror(3)](https://manpages.debian.org/dlerror\(3\))_ and _[gdbm_strerror](https://manpages.debian.org/gdbm\(3\))_ in [`ctypes`](https://docs.python.org/3/library/ctypes.html#module-ctypes "ctypes: A foreign function library for Python.") and [`dbm.gnu`](https://docs.python.org/3/library/dbm.html#module-dbm.gnu "dbm.gnu: GNU database manager \(Unix\)") functions respectively. Patch by Bénédikt Tran.
  * [gh-127873](https://github.com/python/cpython/issues/127873): When `-E` is set, only ignore `PYTHON_COLORS` and not `FORCE_COLOR`/`NO_COLOR`/`TERM` when colourising output. Patch by Hugo van Kemenade.
  * [gh-127870](https://github.com/python/cpython/issues/127870): Detect recursive calls in ctypes `_as_parameter_` handling.
