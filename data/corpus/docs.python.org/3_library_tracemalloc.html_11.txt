Title: tracemalloc — Trace memory allocations
URL: https://docs.python.org/3/library/tracemalloc.html
Summary: The document provides details on the `tracemalloc` module in Python, which is used for tracing memory allocations. It includes a table of contents outlining examples, such as displaying the top memory allocations, computing differences, retrieving memory block tracebacks, and recording memory sizes, as well as information on the API and its functions.
