Title: Importing Modules in Python
URL: https://docs.python.org/3/library/modules.html
Summary: This section covers various modules to import other Python modules and customize the import process. Key modules include `zipimport` for importing from Zip archives, `pkgutil` for package utilities, `modulefinder` for identifying modules used by scripts, and `runpy` for module execution.
---

# Importing Modules[¶](https://docs.python.org/3/library/modules.html#importing-modules "Link to this heading")
The modules described in this chapter provide new ways to import other Python modules and hooks for customizing the import process.
