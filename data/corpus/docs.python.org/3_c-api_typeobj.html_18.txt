Title: Setting Attributes in Python C API
URL: https://docs.python.org/3/c-api/typeobj.html
Summary: This section discusses the handling of attribute deletion by supporting a NULL value in the context of setting attributes. It highlights the convenience of using `PyObject_GenericSetAttr()` for attribute management. Additionally, it notes the inheritance of the `tp_setattr` and `tp_setattro` fields by subtypes.

In addition, setting _value_ to `NULL` to delete an attribute must be supported. It is usually convenient to set this field to [`PyObject_GenericSetAttr()`](https://docs.python.org/3/c-api/object.html#c.PyObject_GenericSetAttr "PyObject_GenericSetAttr"), which implements the normal way of setting object attributes.
**Inheritance:**
Group: [`tp_setattr`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_setattr "PyTypeObject.tp_setattr"), [`tp_setattro`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_setattro "PyTypeObject.tp_setattro")
This field is inherited by subtypes together with [`tp_setattr`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_setattr "PyTypeObject.tp_setattr"): a subtype inherits both [`tp_setattr`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_setattr "PyTypeObject.tp_setattr") and [`tp_setattro`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_setattro "PyTypeObject.tp_setattro") from its base type when the subtype’s [`tp_setattr`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_setattr "PyTypeObject.tp_setattr") and [`tp_setattro`](https://docs.python.org/3/c-api/typeobj.html#c.PyTypeObject.tp_setattro "PyTypeObject.tp_setattro") are both `NULL`.
**Default:**
[`PyBaseObject_Type`](https://docs.python.org/3/c-api/structures.html#c.PyBaseObject_Type "PyBaseObject_Type") uses [`PyObject_GenericSetAttr()`](https://docs.python.org/3/c-api/object.html#c.PyObject_GenericSetAttr "PyObject_GenericSetAttr").
