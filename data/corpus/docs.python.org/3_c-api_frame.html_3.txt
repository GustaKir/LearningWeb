Title: PyUnstable_InterpreterFrame_GetLine Function
URL: https://docs.python.org/3/c-api/frame.html
Summary: The function int PyUnstable_InterpreterFrame_GetLine retrieves the currently executing line number from a PyInterpreterFrame, returning -1 if no line number exists. This is part of an unstable API introduced in version 3.12, which may change unexpectedly in future minor releases.
---

intPyUnstable_InterpreterFrame_GetLine(struct[_PyInterpreterFrame](https://docs.python.org/3/c-api/frame.html#c._PyInterpreterFrame "_PyInterpreterFrame")*frame);[¶](https://docs.python.org/3/c-api/frame.html#c.PyUnstable_InterpreterFrame_GetLine "Link to this definition")
    
_This is[ Unstable API](https://docs.python.org/3/c-api/stable.html#unstable-c-api). It may change without warning in minor releases._
Return the currently executing line number, or -1 if there is no line number.
Added in version 3.12.
