Title: PyCodec_Decode Function
URL: https://docs.python.org/3/c-api/codec.html
Summary: The PyCodec_Decode function is a part of Python's C API for generic codec-based decoding. It takes an object, encoding, and error handling parameters, returning a new reference. It raises a LookupError if no encoder is found.
---

[PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*PyCodec_Decode([PyObject](https://docs.python.org/3/c-api/structures.html#c.PyObject "PyObject")*object, constchar*encoding, constchar*errors)[¶](https://docs.python.org/3/c-api/codec.html#c.PyCodec_Decode "Link to this definition")
    _Return value: New reference.__Part of the[ Stable ABI](https://docs.python.org/3/c-api/stable.html#stable)._
Generic codec based decoding API.
_object_ is passed through the decoder function found for the given _encoding_ using the error handling method defined by _errors_. _errors_ may be `NULL` to use the default method defined for the codec. Raises a [`LookupError`](https://docs.python.org/3/library/exceptions.html#LookupError "LookupError") if no encoder can be found.
## Codec lookup API[¶](https://docs.python.org/3/c-api/codec.html#codec-lookup-api "Link to this heading")
In the following functions, the _encoding_ string is looked up converted to all lower-case characters, which makes encodings looked up through this mechanism effectively case-insensitive. If no codec is found, a [`KeyError`](https://docs.python.org/3/library/exceptions.html#KeyError "KeyError") is set and `NULL` returned.
